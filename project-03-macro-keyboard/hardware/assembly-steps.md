# 组装步骤 | Assembly Steps

## 概述 | Overview

本文档指导如何将焊接完成的 PCB 组装为成品宏键盘，包括轴体安装、键帽安装和外壳组装。

This document guides you through assembling the soldered PCB into a finished macro keyboard, including switch installation, keycap placement, and case assembly.

---

## 组装前检查 | Pre-assembly Checklist

在开始组装之前，确认以下事项：

Before starting assembly, verify the following:

| 检查项 Item | 标准 Criteria | 状态 Status |
|---|---|---|
| 所有元件已焊接 | 无遗漏 | [ ] |
| 焊接质量合格 | 无虚焊、桥连 | [ ] |
| 短路测试通过 | VCC 与 GND 不短路 | [ ] |
| 通电测试正常 | USB 识别，电流正常 | [ ] |
| 固件已刷入 | 键盘可被识别为 HID 设备 | [ ] |
| 3D 打印外壳已到货 | 无明显打印缺陷 | [ ] |

---

## 第一步：安装热插拔轴体 | Step 1: Install Switches

**操作步骤:**

1. **确认轴座方向**：热插拔轴座有两个孔，分别对应轴体的两个引脚
2. **对准轴体**：轴体底部有两个金属引脚和一个塑料定位柱
3. **插入轴体**：将轴体垂直对准轴座孔位，均匀用力按下
4. **听到"咔嗒"声**：表示轴体已完全插入

```
轴体安装示意图：

        键帽（最后装）
         ┌───┐
         │   │
    ┌────┤   ├────┐
    │    │   │    │  ← 轴体外壳
    │  ┌─┴───┴─┐  │
    │  │  引脚  │  │  ← 金属引脚插入轴座
    └──┤       ├──┘
       │  轴座  │     ← 热插拔轴座（焊接在 PCB 上）
       └───┬───┘
           │
          PCB
```

**注意事项 Notes:**
- 安装前检查轴体引脚是否弯曲，弯曲的引脚无法插入轴座
- 如果插入困难，不要强行按压，检查引脚方向
- 可以先安装 4-5 个轴体测试功能，确认正常后再全部安装

---

## 第二步：安装旋钮帽 | Step 2: Install Encoder Knob

**操作步骤:**

1. 确认 EC11 旋钮的轴是 D 型（平面+圆形）
2. 将旋钮帽对准 D 型轴，按下
3. 旋钮帽应牢固不松动

**注意事项 Notes:**
- 不同品牌的旋钮帽孔径可能不同，确保匹配
- 旋钮帽不应过高，以免与键帽干涉

---

## 第三步：安装 OLED 显示屏 | Step 3: Install OLED Display

**操作步骤:**

1. 如果使用排针连接：将 OLED 模块插入 PCB 上的排针座
2. 如果直焊：OLED 已在 Day 10 焊接到 PCB 上
3. 确认 OLED 方向正确（VCC/GND 不要接反）

**安装位置建议:**
- OLED 通常位于键盘左上角或顶部中央
- 倾斜安装（约 15-30 度角）可提高可读性
- 可使用 3D 打印支架固定倾斜角度

---

## 第四步：安装键帽 | Step 4: Install Keycaps

**操作步骤:**

1. 将键帽对准轴体顶部的十字柱
2. 均匀用力按下，直到键帽完全就位
3. 依次安装全部 16 个键帽

```
键帽安装示意：

  ┌───┐  ┌───┐  ┌───┐  ┌───┐
  │ 7 │  │ 8 │  │ 9 │  │ / │    ← 键帽
  └─┬─┘  └─┬─┘  └─┬─┘  └─┬─┘
    │      │      │      │
   [轴]   [轴]   [轴]   [轴]    ← 轴体
    │      │      │      │
  ══╧══════╧══════╧══════╧══    ← PCB
```

**键帽排列建议:**
- 可以使用键帽上的刻字来标识功能
- 也可以使用空白键帽 + 标签贴纸
- VIA 改键后记得更新键帽标识

---

## 第五步：组装外壳 | Step 5: Assemble Case

**操作步骤:**

1. **检查外壳**：
   - 确认 PCB 能放入外壳内腔
   - 确认 USB 开口位置正确
   - 确认螺丝柱/卡扣位置匹配

2. **放入 PCB**：
   - 将组装好的 PCB 放入外壳底部
   - USB 接口对准外壳开口
   - 确认所有元件都在外壳内部

3. **固定 PCB**：
   - 方案 A：M3 螺丝固定（推荐）
   - 方案 B：卡扣固定
   - 方案 C：双面胶/硅胶垫（简易方案）

4. **安装外壳上盖**（如果有）：
   - 上盖通常有按键开口
   - 确认按键可以从开口中伸出

5. **安装防滑垫**：
   - 在外壳底部四角粘贴硅胶防滑垫
   - 防止键盘在桌面上滑动

```
外壳组装截面图：

   ┌────────────────────────┐
   │   ┌──┐ ┌──┐ ┌──┐ ┌──┐│  ← 上盖（按键开口）
   │   │键│ │键│ │键│ │键││
   │   └┬─┘ └┬─┘ └┬─┘ └┬─┘│
   │    │    │    │    │   │
   │   [轴] [轴] [轴] [轴] │  ← 轴体
   │    │    │    │    │   │
   │  ══╧════╧════╧════╧══ │  ← PCB
   │                        │
   │  [OLED]     [旋钮]    │  ← 外设
   │                        │
   └────────────────────────┘  ← 底壳
       ○               ○       ← 防滑垫
```

## 最终检查 | Final Inspection

| 检查项 Item | 方法 Method | 通过标准 Criteria |
|---|---|---|
| 外观检查 | 目视 | 键帽整齐，外壳无缝隙 |
| 按键手感 | 逐一按键 | 所有按键行程一致，无卡键 |
| 旋钮操作 | 旋转 + 按压 | 旋转有刻度感，按下有反馈 |
| OLED 显示 | 目视 | 显示清晰，角度可读 |
| RGB 灯效 | 通电检查 | 灯光均匀，颜色正确 |
| USB 连接 | 插入电脑 | 设备管理器识别为键盘 |
| 功能测试 | 记事本测试 | 全部按键输入正确 |

## 常见组装问题 | Common Assembly Issues

| 问题 Problem | 原因 Cause | 解决方法 Solution |
|---|---|---|
| 轴体插不进去 | 引脚弯曲 | 用镊子矫正引脚 |
| 键帽松动 | 轴体十字柱磨损 | 更换轴体 |
| 键帽太紧 | 键帽内径偏小 | 轻轻旋转按入，不要强行锤击 |
| PCB 在外壳中晃动 | 固定不牢 | 添加硅胶垫片填充间隙 |
| USB 线插不进去 | 外壳开口偏小 | 用刀片扩大开口 |
| 按键不触发 | 轴体未完全插入 | 拔出重新插入，确保到位 |
