## 三、学习教程资源

### 硬件设计

#### 键盘矩阵扫描原理
- **核心知识点**：将按键排列成行(Row) x 列(Column)的矩阵，通过逐行扫描检测按键状态
  - 每个键位串联一个**二极管**防止鬼键(Ghosting)
  - 典型扫描频率：100Hz-1000Hz
  - 消抖时间：5-20ms
- **推荐学习资源**：
  - QMK源码中的 `matrix.c` 文件（https://github.com/qmk/qmk_firmware）
  - 知乎搜索"机械键盘矩阵扫描原理"
  - CSDN搜索"键盘矩阵扫描 原理 教程"

### 固件开发

#### QMK 固件学习路径
1. **环境搭建**：安装 QMK MSYS（Windows）或设置 Linux 环境
2. **基础编译**：使用 `qmk compile` 命令编译固件
3. **自定义键盘**：创建自己的键盘定义文件（keyboard.json / config.h / rules.mk）
4. **键位映射**：编写 keymap.c 定义按键功能
5. **进阶功能**：添加RGB、OLED、编码器支持
6. **VIA/Vial集成**：实现运行时改键

### 通信协议

#### I2C通信（OLED屏幕通信）
- **典型应用**：连接SSD1306 OLED屏幕（0.96寸 128x64）
- **关键参数**：
  - SCL（时钟线）+ SDA（数据线）= 2根线
  - 标准速率100kHz，快速模式400kHz
  - OLED地址：通常为0x3C或0x3D
- **学习资源**：
  - CSDN搜索"I2C协议 详解 教程"
  - 知乎搜索"I2C通信协议 入门"
  - QMK官方OLED驱动文档：https://docs.qmk.fm/#/feature_oled_driver

#### SPI通信（高速外设通信）
- **典型应用**：高速OLED、TFT屏幕、SPI Flash存储
- **关键信号**：SCLK（时钟）、MOSI（主机出）、MISO（主机入）、CS（片选）
- **学习资源**：
  - CSDN搜索"SPI协议 详解 教程"

#### USB HID协议
- **核心概念**：
  - HID = Human Interface Device（人机接口设备）
  - Report Descriptor（报告描述符）定义数据格式
  - 使用中断传输（Interrupt Transfer）
  - Usage Page定义设备类别
- **键盘HID报告格式**：
  - 8字节标准报告：[修饰键, 保留, 键1, 键2, 键3, 键4, 键5, 键6]
  - 支持同时6键无冲突
- **学习资源**：
  - USB官方规范：https://www.usb.org/hid
  - 《圈圈教你玩USB》（书籍）
  - CSDN搜索"USB HID 协议详解 入门"
  - QMK源码中的USB HID实现

| **方案三（推荐进阶）** | STM32F401/F072 | ARM内核，性能强，QMK支持好 | 10-20元 |
| **方案四（推荐学习）** | RP2040 | Raspberry Pi Pico核心，Python支持 | 10-15元 |

## 五、学习路线图建议

### 第一阶段：理论学习（1-2周）
- [ ] 学习键盘矩阵扫描原理
- [ ] 了解I2C/SPI通信协议基础
- [ ] 了解USB HID协议原理
- [ ] 熟悉QMK固件架构

### 第二阶段：工具准备（1周）
- [ ] 安装嘉立创EDA或KiCad
- [ ] 搭建QMK编译环境（QMK MSYS）
- [ ] 安装STM32CubeMX（若使用STM32方案）

### 第三阶段：PCB设计（2-3周）
- [ ] 绘制键盘原理图（矩阵 + 主控 + 外设）
- [ ] 布局PCB（注意走线规则）
- [ ] 生成Gerber文件并送嘉立创打样

### 第四阶段：固件开发（2-3周）
- [ ] 编写键盘矩阵扫描代码
- [ ] 实现USB HID键盘功能
- [ ] 添加OLED屏幕显示
- [ ] 添加旋钮编码器支持
- [ ] 添加RGB灯效

### 第五阶段：组装调试（1-2周）
- [ ] 焊接元器件到PCB
- [ ] 刷入固件并调试
