### 1. 技术实现 (40 分) | Technical Implementation

#### 1.1 核心示波器功能 (25 分)

| 分数 | 标准 |
|------|------|
| 21-25 | TFT 实时显示波形，触发稳定，时基可调，频率测量准确 (<5% 误差)，自动量程正常 |
| 16-20 | 波形显示正常，基本触发可用，时基可调，频率测量大致准确 |
| 11-15 | 能在 TFT 上显示波形，但触发不稳定或频率测量有明显误差 |
| 6-10 | 能采集 ADC 数据并输出，但波形显示不完整 |
| 0-5 | 未能实现基本数据采集 |
