### 任务 12.2: PPT 准备 (60 分钟)

**PPT 结构建议（5 分钟）：**

| 页码 | 内容 | 时间 |
|------|------|------|
| 1 | 封面：项目名称、姓名、日期 | 15秒 |
| 2 | 项目简介：做什么、为什么做 | 30秒 |
| 3 | 技术架构：系统框图 | 45秒 |
| 4 | 硬件设计：原理图、PCB、实物照片 | 45秒 |
| 5 | 软件设计：核心算法（触发、测频） | 45秒 |
| 6 | 演示截图/视频：波形显示、功能演示 | 45秒 |

**PPT 制作要点：**
- 图片为主，文字为辅
- 每页不超过 5 行文字
- 准备一个 30 秒的演示视频作为备用
- 提前演练，控制在 5 分钟内

### 任务 12.2b: 创建项目展示网站 (60 分钟) | Create Project Demo Website

**你必须创建一个在线可访问的项目展示网站，这是最终评分的重要组成部分（15分）。**

**You must create an online-accessible project demo website. This is a critical part of your final grade (15 points).**

**步骤 Step-by-step：**

1. **将项目文档转化为网站内容**
   - 整理 README.md 中的项目介绍、硬件清单、软件架构等内容
   - 收集 12 天中拍摄的实物照片、波形截图、接线图
   - 撰写 12 天学习心得与总结
   - 整理 B 站参考视频链接列表

2. **使用 GitHub Pages 部署（推荐）**
   - 在仓库中创建 `docs/` 文件夹，放入 `index.html` 和相关资源
   - 或在仓库 Settings > Pages 中选择 `gh-pages` 分支部署
   - 也可使用 Vercel 或 Netlify 等其他免费平台

3. **网站必须包含以下内容模块：**
   - [ ] 项目介绍（名称、目标、核心功能）
   - [ ] 硬件展示（BOM 清单、实物照片、接线图、电路原理图）
   - [ ] 软件架构（系统框图、模块说明、关键代码片段）
   - [ ] 演示视频或 GIF（展示示波器测量实际信号的效果）
   - [ ] 性能指标（实测带宽、采样率、波形截图）
   - [ ] 12 天学习心得与总结
   - [ ] B 站参考视频链接列表

4. **提交 URL**
   - 在最终展示前，将网站 URL 提交给老师
   - URL 格式示例：`https://你的用户名.github.io/project-name/`

**技术建议 Technical Tips：**
- 最简方案：一个 `index.html` 文件 + CSS 样式即可
- 可使用 GitHub Pages 的 Jekyll 主题快速生成
- 推荐工具：Markdown + 静态网站生成器（如 VitePress、Hugo）

## 上午/下午: 项目展示 | Morning/Afternoon: Project Presentation

### 任务 12.3: 现场展示 (每组 8 分钟)

**展示流程：**

1. **PPT 演示 (5 分钟)**
   - 讲解项目背景和技术原理
   - 展示设计过程和关键决策
   - 展示最终成果

2. **现场演示 (2 分钟)**
   - 用信号发生器输入标准信号
   - 演示示波器的核心功能：
     - 波形显示
     - 频率测量
     - 时基切换
     - 触发模式切换
   - 演示信号发生器功能（如果实现了）

3. **评委提问 (1 分钟)**
   - 评委可能问的问题：
     - "ADC 的采样率是如何确定的？"
     - "触发是怎么实现的？"
     - "如果要提高带宽，你会怎么做？"
     - "这个项目最大的挑战是什么？"
     - "如果重新做，你会改进什么？"

**展示技巧：**
- 保持自信，语速适中
- 先展示能工作的功能，再讲原理
- 如果某个功能失败了，不要慌张，讲讲你打算怎么解决
- 准备好回答"为什么"——评委最关心的是你的理解深度

## 课程回顾 | Course Recap

### 12 天学习之旅 | 12-Day Learning Journey

```
Phase 1 (Day 1-4): 基础
  Day 1: 项目启动 ─── 了解示波器，搭建开发环境
  Day 2: 运放电路 ─── 信号调理，放大和偏置
  Day 3: ADC+DMA  ─── 模数转换，高速采样
  Day 4: 数据采集 ─── 完整采集链路，PC 端验证

Phase 2 (Day 5-8): 核心功能
  Day 5: 触发逻辑 ─── 边沿检测，波形稳定
  Day 6: TFT 显示 ─── SPI 驱动，波形渲染
  Day 7: 时基控制 ─── 采样率切换，水平缩放
  Day 8: 频率测量 ─── 参数测量，自动量程

Phase 3 (Day 9-12): 拓展与展示
  Day 9: DDS 信号发生器 ─── AD9833 驱动，波形生成
  Day 10: PCB 设计     ─── 原理图，PCB 布局，焊接
  Day 11: 外壳设计     ─── 3D 建模，系统集成
  Day 12: 最终展示     ─── 演示，复盘，庆祝
```

### 进一步学习建议 | Next Steps

如果你想继续深入学习，建议：

1. **提高带宽：** 使用外部高速 ADC (如 AD9280, 32Msps)
2. **双通道：** 增加第二路 ADC 通道，实现双通道示波器
3. **FFT 频谱分析：** 在 STM32 上实现快速傅里叶变换，显示频谱图
4. **上位机通信：** 通过 USB/蓝牙将波形数据发送到 PC 端软件
5. **嵌入式操作系统：** 学习 FreeRTOS，用 RTOS 管理多任务
6. **FPGA 示波器：** 使用 FPGA 实现更高速的采样和处理
