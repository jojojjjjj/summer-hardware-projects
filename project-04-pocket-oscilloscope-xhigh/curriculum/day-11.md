## 上午: 外壳设计 | Morning: Case Design

### 为什么要学这个? | Why Learn This?

一个完整的电子产品不仅需要电路，还需要外壳。好的外壳设计不仅让产品美观，更重要的是：
- **保护电路：** 防止灰尘、水滴、短路
- **便于使用：** 合理的按键和接口位置
- **便携性：** 口袋大小，方便携带
- **散热：** 确保设备长时间工作不发热

3D 打印是快速原型制作的最佳方式。从设计到实物只需几个小时，而且可以反复迭代修改。

A complete electronic product needs not just a circuit but also a case. Good case design not only makes the product look nice, but more importantly: protects the circuit, provides easy-to-use buttons and ports, ensures portability, and manages heat. 3D printing is the best way for rapid prototyping -- from design to physical object in just hours.

### 任务 11.1: 外壳设计参数 (30 分钟)

**外壳尺寸规划：**

```
内部空间需要容纳：
  - PCB / 面包板 (50mm x 80mm)
  - TFT 显示屏 (约 35mm x 28mm)
  - STM32 蓝pill板 (53mm x 22mm)
  - AD9833 模块 (约 15mm x 15mm)
  - 运放电路区域
  - 按键 (3~4 个)
  - BNC/输入接口

外壳外部尺寸（参考）：
  - 长度：90~100mm
  - 宽度：55~60mm
  - 高度：20~25mm

开口：
  - TFT 显示屏窗口
  - USB 供电口
  - 信号输入口 (BNC 或排针)
  - 按键孔位
```

**设计工具选择：**

| 工具 | 特点 | 推荐度 |
|------|------|--------|
| TinkerCAD | 在线免费，极其简单 | 最推荐新手 |
| Fusion 360 | 专业级，学生免费 | 进阶 |
| 嘉立创 EDA 3D | 与 PCB 联动 | PCB 相关 |
| OpenSCAD | 编程建模 | 极客向 |

### 任务 11.2: TinkerCAD 外壳建模 (60 分钟)

**步骤 Steps:**

1. 访问 https://www.tinkercad.com/ 并注册/登录
2. 新建 3D 设计项目
3. **创建外壳底座：**
   - 拖入一个 "Box" 形状
   - 尺寸：95mm x 58mm x 10mm
   - 设为"中空"（挖空内部）
   - 壁厚：2mm
4. **创建外壳顶盖：**
   - 复制底座形状
   - 高度设为 5mm
   - 添加卡扣结构（可选）
5. **添加开口：**
   - TFT 窗口：35mm x 28mm，在顶面
   - USB 口：10mm x 5mm，在侧面
   - 输入接口孔：6mm 圆孔，在侧面
   - 按键孔：5mm 圆孔 x 3~4 个，在顶面或侧面
6. **导出：** 导出为 STL 格式

**预期结果 Expected Result:**
- 完成外壳底座和顶盖的 3D 模型
- 各开口位置与硬件匹配
- 导出 STL 文件

**常见问题 Common Issues:**
- "开口位置对不上" -- 先用卡尺测量实际硬件尺寸，再建模
- "打印出来太薄/太厚" -- 壁厚 2mm 比较合适，太薄易碎，太厚浪费材料
- "不知道怎么挖孔" -- TinkerCAD 的 "Hole" 功能可以做减法运算

### 任务 11.3: 3D 打印 (45 分钟)

**3D 打印基本流程：**

1. **切片：** 使用 Cura 或 PrusaSlicer 将 STL 文件转为 G-code
   - 层高：0.2mm (标准质量)
   - 填充率：20%
   - 壁厚：3 层
   - 材料：PLA (最常用)
   - 无需支撑（如果没有悬空结构）
2. **打印：** 将 G-code 文件拷贝到 SD 卡，插入 3D 打印机
3. **等待：** 一个外壳大约需要 1~2 小时
4. **后处理：** 去除支撑、打磨粗糙面

**如果没有 3D 打印机：**
- 方案 A：使用学校实验室的 3D 打印机
- 方案 B：淘宝搜索"3D 打印代打"，上传 STL 文件，约 20~30 元
- 方案 C：使用硬纸板/亚克力手工制作（应急方案）

## 下午: 系统集成与调试 | Afternoon: System Integration & Debugging

### 任务 11.4: 硬件装配 (60 分钟)

**装配步骤（如果是面包板方案）：**

1. 将面包板裁剪到合适的尺寸
2. 安装 STM32 蓝pill板（用双面胶或铜柱固定）
3. 安装 TFT 显示屏（对准窗口开口）
4. 安装按键（从外壳按键孔穿出）
5. 连接所有杜邦线
6. 安装 USB 供电线（从侧面开口引出）
7. 安装信号输入接口

**装配步骤（如果有 PCB）：**

1. 焊接所有元件到 PCB 上
   - 先焊接低矮的贴片元件 (电阻、电容)
   - 再焊接 IC 座和排针
   - 最后焊接大元件 (电解电容、连接器)
2. 安装 PCB 到外壳底座
3. 安装 TFT 显示屏
4. 安装按键帽

**装配检查清单：**

- [ ] 所有元件安装到位
- [ ] 连接线无松动
- [ ] TFT 窗口对齐
- [ ] 按键操作灵活
- [ ] USB 口可正常插入
- [ ] 无短路隐患

### 任务 11.5: 整机系统调试 (60 分钟)

**调试步骤：**

1. **上电测试：**
   - 接 USB 电源，确认 LED 指示灯亮
   - 确认 TFT 屏幕正常显示
   - 测量 3.3V 和 5V 电源电压

2. **示波器功能测试：**
   - 连接信号发生器输出到输入口
   - 输入 1kHz 正弦波
   - 确认 TFT 显示波形
   - 测试时基切换
   - 测试频率测量
   - 测试自动量程

3. **信号发生器功能测试：**
   - 按键切换频率和波形
   - 用万用表/示波器测量输出
   - 验证自收自发功能

4. **压力测试：**
   - 长时间运行 30 分钟，确认不发热、不崩溃
   - 反复切换功能，确认稳定

**整机性能指标验证：**

| 指标 | 目标 | 测量方法 |
|------|------|---------|
| 带宽 | > 100kHz | 输入不同频率正弦波，观察最高清晰频率 |
| 采样率 | 1Msps | 已通过定时器配置验证 |
| 频率测量精度 | < 5% | 对比信号发生器显示值 |
| 电压测量精度 | < 10% | 对比万用表测量值 |
| 波形刷新率 | > 10fps | 观察波形更新流畅度 |
| 功耗 | < 500mA | USB 电流表测量 |
