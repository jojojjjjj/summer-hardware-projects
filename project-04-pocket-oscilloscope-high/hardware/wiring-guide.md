# 接线与电路指南 | Wiring & Circuit Guide

## 概述 | Overview

本文档描述 TinyAWG 口袋信号发生器的电路连接和信号流程。本项目使用定制 PCB（非面包板），所有接线已集成在 PCB 设计中。本文档帮助你理解电路架构，便于调试和修改。

This document describes the circuit connections and signal flow of the TinyAWG pocket signal generator. This project uses a custom PCB (not breadboard), with all connections integrated into the PCB design. This guide helps you understand the circuit architecture for debugging and modification.

### 重要电源注意事项

| 注意事项 | 说明 |
|---------|------|
| 外部电源必须 9-12V | 5V 只能充电，不能开机使用 |
| 接入电源时电池必充电 | 请确保使用 9-12V 电源适配器 |
| 模拟电源 ±6.5V | 输出不会超过此范围 |
| 无输出保护 | 使用时需谨慎，避免短路 |
| USB 串口需要分路器 | 电源和串口共用 Type-C，需分路器 |

## ZYNQ 核心板连接 | ZYNQ Core Board Connections

### 板对板连接器引脚分配

ZYNQ7010 核心板通过板对板连接器与底板连接。主要信号分组：

| 信号组 | 方向 | 说明 |
|--------|------|------|
| PS DDR3 | 双向 | 512MB DDR3 内存 (核心板内置) |
| PL 时钟 | 输入 | 200MHz 差分时钟 |
| PL 数据总线 [13:0] | 输出 | 14-bit 并行数据到 AD9744 |
| PL 时钟输出 | 输出 | DAC 时钟 |
| PS SPI (SPI0) | 主出 | DAC8562 片选/时钟/数据 |
| PS I2C (I2C0) | 主出 | MAX17048, FT6336 |
| PS I2C (I2C1) | 主出 | 备用 |
| PS GPIO | 双向 | 继电器控制, LED, 按键 |
| PS UART | 双向 | USB 串口通信 |
| PS LCD | 输出 | 并口 RGB 信号到 LCD |
| JTAG | 双向 | 调试接口 |

### AD9744 DAC 接口

| 信号 | 方向 | 说明 |
|------|------|------|
| D[13:0] | PL→DAC | 14-bit 并行数据，2's complement 格式 |
| CLK | PL→DAC | DAC 采样时钟 (200MHz) |
| SLEEP | PL→DAC | 省电模式控制 (低电平有效) |

### DAC8562 辅助 DAC (SPI)

| 信号 | 方向 | ZYNQ 引脚 | 说明 |
|------|------|----------|------|
| SCLK | PS→DAC | SPI0_SCLK | SPI 时钟 |
| DIN | PS→DAC | SPI0_MOSI | SPI 数据 |
| SYNC | PS→DAC | GPIO | 片选 (低电平有效) |
| LDAC | PS→DAC | GPIO | 同步更新 (低电平有效) |

### OPA2673 输出级

- 参考设计：DHO914S AWG 电路
- 运放 A：I/V 转换（电流→电压）
- 运放 B：放大 + 偏置叠加
- 增益由 DAC8562 CH-A 控制
- 偏置由 DAC8562 CH-B 控制

|------|------|----------|------|
| SDA | 双向 | I2C0_SDA | I2C 数据 |
| SCL | PS→FT6336 | I2C0_SCL | I2C 时钟 |
| INT | FT6336→PS | GPIO | 触摸中断 |
| RST | PS→FT6336 | GPIO | 复位 |

| 信号 | 方向 | ZYNQ 引脚 | 说明 |

| SDA | 双向 | I2C0_SDA | I2C 数据 (与 FT6336 共享总线) |
| SCL | PS→MAX17048 | I2C0_SCL | I2C 时钟 (与 FT6336 共享总线) |

### WS2812 RGB LED

| 信号 | 方向 | ZYNQ 引脚 | 说明 |
|------|------|----------|------|
| DIN | PS→LED | GPIO | 单线数据 (NRZ 编码) |

### 继电器控制

| 信号 | 方向 | ZYNQ 引脚 | 说明 |
|------|------|----------|------|
| RELAY1 | PS→继电器 | GPIO | 输出通道 1 切换 |
| RELAY2 | PS→继电器 | GPIO | 输出通道 2 切换 |

### USB Type-C (电源+串口)

| 信号 | 说明 |
|------|------|
| VBUS | 5V/12V 电源输入 (经 CH221K PD 诱骗) |
| D+/D- | USB 2.0 数据 (经 USB-UART 桥接) |
| GND | 地线 |

> **重要提示**：USB Type-C 同时用于供电和串口通信。如果需要连接电脑串口调试，必须使用电源数据分路器。参见 `troubleshooting.md` 了解详情。

## 安全警告 | Safety Warnings

1. **电源警告**：接入外部电源时，电池一定会进行充电。必须使用 9-12V 电源适配器。如果只有 5V，只能用于充电，不要开机。
2. **输出无保护**：由于 PCB 面积限制，输出端没有内置保护电路。避免短路和过载。
3. **高压警告**：模拟电源轨为 ±6.5V，输出最高可达 ±5V。注意安全操作。
4. **静电防护**：AD9744、OPA2673 等芯片对静电敏感。操作时请佩戴防静电手环。
5. **电池安全**：使用带保护板的锂电池，避免过充、过放、短路。
