## 上午 (3h) | Morning Session

### 1. 电路集成入外壳 (2h)

**安装顺序**（从内到外）：

```
安装步骤：

Step 1: 固定 ESP32-C3 主控板
  - 将 ESP32-C3 放入框架背面中央
  - Type-C 接口朝下（方便充电）
  - 用铜线或热熔胶固定 4 个角
  - 确保天线朝上（WiFi 信号更好）

Step 2: 连接 TFT 显示屏
  - 将 TFT 屏幕从内侧推入显示窗口
  - 用少量热熔胶在 4 个角固定
  - 确保屏幕居中，不歪斜
  - 连接 SPI 线路（8 线）

Step 3: 安装 SHT31-D 传感器
  - 将传感器固定在框架顶部
  - 传感器感应窗口朝外（方便空气流通）
  - 连接 I2C 线路（4 线）
  - 注意：传感器要远离 ESP32-C3（避免芯片发热影响读数）

Step 4: 安装锂电池
  - 将电池放在 ESP32-C3 下方
  - 用双面胶或铜线固定
  - 连接电池线到充电模块

Step 5: 安装充电模块
  - Type-C 充电模块固定在框架底部
  - USB 接口朝下（通过底座出口）
  - 连接充电线到电池

Step 6: 安装触摸开关
  - TTP223 触摸模块固定在框架外侧
  - 触摸面朝外
  - 连接信号线到 ESP32-C3

Step 7: 安装 LED 指示灯
  - 将 LED 穿过铜线框架
  - LED 头露出框架表面
  - 用热熔胶固定

Step 8: 连接太阳能板（如适用）
  - 将太阳能板放在支架上
  - 用少量热熔胶固定
  - 连接太阳能板线到充电控制器
```

**接线最终检查表**：

```
接线检查清单：

ESP32-C3 连接：
  □ GPIO0  (SDA)  ←→ SHT31-D SDA
  □ GPIO1  (SCL)  ←→ SHT31-D SCL
  □ GPIO2  (TOUCH) ←→ TTP223 SIG
  □ GPIO3  (LED)  ←→ LED 正极
  □ GPIO4  (SCLK) ←→ ST7735 SCLK
  □ GPIO5  (MOSI) ←→ ST7735 MOSI
  □ GPIO6  (CS)   ←→ ST7735 CS
  □ GPIO7  (DC)   ←→ ST7735 DC
  □ GPIO8  (LED2) ←→ 状态 LED
  □ GPIO10 (RST)  ←→ ST7735 RST
  □ 3V3            ←→ 所有模块 VCC
  □ GND            ←→ 所有模块 GND

电源连接：
  □ 电池 (+)      ←→ 充电模块 BAT+
  □ 电池 (-)      ←→ 充电模块 BAT-
  □ 充电模块 OUT+ ←→ ESP32-C3 VCC (或 5V 引脚)
  □ 充电模块 OUT- ←→ ESP32-C3 GND
  □ 太阳能板 (+)  ←→ 充电控制器 IN+
  □ 太阳能板 (-)  ←→ 充电控制器 IN-

总计约 20+ 根线 -- 整齐排列是关键！
```

### 2. 底座固定与走线整理 (1h)

**底座安装**：

```
最终底座安装：

1. 将所有线路从立柱中心穿过
   - 避免线路裸露在外
   - 太阳能板线沿支架走

2. 将立柱插入底座
   - 用热熔胶固定
   - 确保立柱垂直

3. USB 线出口
   - 在底座侧面留出 USB Type-C 线出口
   - 线不要绷太紧

4. 整理多余线缆
   - 将过长的线缆用扎带或铜线捆扎
   - 隐藏在框架内部
```

**走线整理技巧**：

```
走线整理方法：

1. 分组捆扎：
   - SPI 线组（8 线）用一段铜线或扎带捆在一起
   - I2C 线组（4 线）另捆一组
   - 电源线单独走

2. 隐藏线路：
   - 尽量将线路安排在框架内侧
   - 从正面看不应看到任何裸露线路

3. 固定点：
   - 每隔 1-2cm 用少量热熔胶固定线路
   - 防止线路晃动或拉扯

4. 留有余量：
   - 在连接点处留 5mm 的余量
   - 方便日后维修或调整
```

### 3. 最终调试 (1.5h)

**全功能测试清单**：

```
功能测试清单（逐项验证）：

核心功能 Core Functions：
  □ TFT 显示屏正常显示（无乱码、无闪烁）
  □ 时钟显示准确（对比手机时间，误差 < 2 秒）
  □ NTP 时间自动同步
  □ 温度读数合理（与教室温度计对比，误差 < 1°C）
  □ 湿度读数合理（误差 < 5%）
  □ WiFi 连接稳定（运行 30 分钟无断线）

电源系统 Power System：
  □ Type-C 充电正常（插入后充电指示灯亮）
  □ 电池电量显示正常
  □ 拔掉 USB 后设备正常运行
  □ 太阳能板充电效果（如有日光条件）

交互功能 Interaction：
  □ 触摸开关短按切换屏幕
  □ 触摸开关长按进入/退出睡眠
  □ LED 指示灯状态正确
  □ 屏幕自动切换（10 秒间隔）

异常处理 Error Handling：
  □ WiFi 断线后自动重连
  □ 传感器读取失败后使用缓存值
  □ NTP 同步失败后继续运行
```

**常见最终问题排查**：

| 问题 Problem | 可能原因 Cause | 解决方法 Solution |
|-------------|--------------|-----------------|
| TFT 显示偏移 | initR 参数不对 | 尝试不同的 TAB 类型 |
| 温度偏高 | ESP32 热量传导 | 增大传感器与 MCU 的距离 |
| WiFi 信号弱 | 天线被铜线遮挡 | 调整天线方向 |
| 触摸不灵敏 | 铜线屏蔽 | 将触摸传感器移到外侧 |
| 电池续航短 | 线路短路或漏电 | 检查所有连接点 |
| 显示闪烁 | SPI 线太长或干扰 | 缩短 SPI 线路 |

### 4. 外观美化与打磨 (1.5h)

**铜线表面处理**：

```
美化方法：

1. 清洁铜线表面
   - 用酒精棉签擦拭焊点残留
   - 去除助焊剂痕迹
   - 用干布擦亮铜线

2. 铜线氧化保护（可选）
   - 涂抹一层透明指甲油
   - 或喷一层透明保护漆
   - 防止铜线变黑氧化

3. 焊点美化
   - 检查所有焊点是否圆润
   - 补焊粗糙的焊点
   - 去除多余的焊锡球

4. 底座美化
   - 在底座上贴标签（项目名称、作者、日期）
   - 用铜线在底座上做简单装饰
```

**细节调整**：

```
最终细节检查：

1. 对称性：
   - 太阳能板支架是否对称？
   - 立柱是否垂直居中？

2. 稳定性：
   - 放在桌面上是否平稳？
   - 轻推是否会倒？

3. 美观度：
   - 从不同角度观察，是否好看？
   - 线路是否隐藏？
   - 焊点是否整洁？

4. 功能性：
   - 所有按钮/开关是否容易操作？
   - USB 口是否容易插拔？
   - 屏幕是否清晰可见？
```
