## 上午 (3h) | Morning Session

### 1. 主体框架搭建 (2h)

**正面面板框架（TFT 显示窗口）**：

```
正面框架尺寸规划：

ST7735 TFT 显示屏尺寸：约 25mm x 18mm（0.96寸显示区域）
加上 PCB 边距：约 30mm x 22mm

框架设计（内框略大于显示屏 PCB）：
  外框：40mm x 30mm
  内框（窗口）：32mm x 24mm

正面视图：
  ┌──────────────────┐
  │  ┌────────────┐  │  ← 铜线外框
  │  │            │  │
  │  │  TFT 屏幕  │  │  ← 显示窗口
  │  │            │  │
  │  └────────────┘  │
  └──────────────────┘
```

**搭建步骤**：

```
Step 1: 弯制正面框架
  - 用 1.0mm 铜线弯制外框矩形（40mm x 30mm）
  - 确保四个角都是直角
  - 在一个角焊接闭合

Step 2: 弯制显示窗口
  - 用 0.7mm 铜线弯制内框矩形（32mm x 24mm）
  - 比 TFT 屏幕略大

Step 3: 连接内外框
  - 用短铜线段将内框固定在外框内部
  - 每个角一个连接点
  - 注意：内框要居中！

Step 4: 检查适配
  - 将 TFT 显示屏放入窗口测试
  - 确保屏幕能稳固卡住
  - 如太紧，微调内框大小
```

**背面框架**：

```
Step 5: 弯制背面框架
  - 与正面框架相同尺寸（40mm x 30mm）
  - 不需要显示窗口

Step 6: 连接正背面
  - 用 4 根铜线段作为侧面连接
  - 长度约 15-20mm（取决于元件厚度）
  - 在 8 个连接点焊接

        正面
       ┌────┐
       │    │
    ═══╪════╪═══  侧面连接线
       │    │
       └────┘
        背面
```

**讨论**：
- 为什么正面框架需要内外两层？（内层固定屏幕，外层提供结构强度）
- 侧面连接线的长度如何确定？（取决于 PCB + 电池 + 传感器的总厚度）
- 如何确保正背面平行？（先焊接对角的两个连接点，再焊另外两个）

### 2. 太阳能板支架 (1h)

**支架设计**：

```
太阳能板支架（左右对称）：

侧视图：
            ┌──────┐
            │太阳能│
            │  板  │
            └──┬───┘
               │  ← 支架铜线
          ┌────┤
          │主体│
          └────┘

支架角度：约 30-45 度倾斜
支架材料：1.0mm 铜线
```

**弯制步骤**：

```
Step 1: 弯制左支架
  - 取 1.0mm 铜线约 15cm
  - 弯制 L 形（一端连接主体，一端支撑太阳能板）
  - 顶部弯制一个小平台（放置太阳能板）

Step 2: 弯制右支架（镜像）
  - 与左支架对称
  - 确保两侧高度一致

Step 3: 焊接到主体
  - 将支架焊接在主体侧面的上下方
  - 注意左右对称
  - 检查角度是否一致
```

## 下午 (4h) | Afternoon Session

### 3. 面板安装与线路整理 (2h)

**电子元件布局规划**：

```
背面元件布局：

┌─────────────────────┐
│                     │
│  [SHT31-D]  [LED]   │  ← 传感器和指示灯（顶部）
│                     │
│  ┌───────────────┐  │
│  │   ESP32-C3    │  │  ← 主控板（中央）
│  │   (Type-C朝下) │  │
│  └───────────────┘  │
│                     │
│  ┌─────┐  [触摸开关] │  ← 电池和触摸开关（底部）
│  │锂电池│           │
│  └─────┘            │
│                     │
└─────────────────────┘
```

**元件固定方法**：

```
方法 1：铜线绑扎法（推荐）
  - 用 0.7mm 铜线将 PCB 绑在框架上
  - 在 4 个角各绑一道
  - 不需要螺丝，可拆卸

方法 2：热熔胶固定法
  - 用少量热熔胶固定元件
  - 优点：快速、牢固
  - 缺点：不易拆卸，高温可能软化

方法 3：焊接固定法
  - 直接将元件焊到铜线框架上
  - 优点：最牢固
  - 缺点：永久固定，无法拆卸
```

**线路走线规划**：

```
线路走线原则：

1. 最短路径：减少线路长度，降低干扰
2. 避免交叉：SPI 和 I2C 线路分开走
3. 固定牢靠：用铜线或热熔胶固定线路
4. 留有余量：留出微调空间，不要绷太紧
5. 美观：尽量将线路隐藏在框架内侧

线路分组：
  - SPI 组（TFT）：SCLK, MOSI, CS, DC, RST, VCC, GND, BL -- 8 线
  - I2C 组（SHT31）：SDA, SCL, VCC, GND -- 4 线
  - 电源组：BAT+, BAT-, USB, CHG -- 4 线
  - 控制组：TOUCH, LED, CHARGE_STAT -- 3 线

建议使用不同颜色的杜邦线区分：
  红色 = VCC/电源
  黑色 = GND/地
  黄色 = SPI 数据
  蓝色 = I2C 数据
  绿色 = 控制信号
```

### 4. 立柱与底座 (1h)

**黄铜管立柱**：

```
立柱安装：

  ┌────────┐
  │  主体  │
  └───┬┬───┘
      ││     ← 两根黄铜管立柱（3mm 外径）
      ││
  ┌───┴┴───┐
  │  底座  │
  └────────┘

安装方法：
  1. 在主体底部中心位置钻两个小孔（或焊接铜线环）
  2. 将黄铜管插入孔中
  3. 用焊锡固定
  4. 在底座上对应位置钻孔
  5. 将管子另一端插入底座
  6. 用热熔胶或焊锡固定

高度：约 5-8cm（让设备看起来像悬浮的卫星）
```

**底座制作**：

```
竹杯垫底座改造：

  1. 取一个直径 6cm 的竹杯垫
  2. 在中心位置标记两个立柱插孔
  3. 用锥子或小钻头钻孔（3mm 直径）
  4. 在底座底部留出 USB Type-C 线的出口
  5. 可选：在底座上刻字或做装饰

    ┌──────────┐
    │  ╔══╗    │  ← 立柱插孔
    │  ║  ║    │
    │  ╚══╝    │
    │          │
    │    ○     │  ← USB 线出口
    └──────────┘
```

### 5. 结构加固与测试 (1h)

**加固措施**：

```
需要加固的部位：

1. 焊点加固：
   - 每个焊点再补一次焊锡
   - 确保焊点饱满、无虚焊

2. 框架对角线加固：
   - 在主体框架的对角线位置加焊一根铜线
   - 防止框架变形为平行四边形

3. 支架根部加固：
   - 太阳能板支架与主体连接处加焊三角支撑
   - 提高抗弯强度

4. 立柱底部加固：
   - 立柱与底座连接处加焊一圈铜线
   - 防止立柱松动
```

**结构强度测试**：

```
测试项：
  □ 轻轻摇晃，框架不松动
  □ 按压太阳能板支架，不弯曲变形
  □ 将电子元件放入框架，空间适配
  □ TFT 屏幕稳固卡在窗口中
  □ 立柱垂直，底座水平
  □ USB 线能正常插入
```
