### 1. 焊接安全与基础 (1h)

**焊接安全操作规程**（必须严格遵守！）：

```
焊接安全 10 条规则：

1. 通风良好：焊接时产生的烟雾有害，保持通风或使用排风扇
2. 戴护目镜：防止焊锡飞溅伤眼
3. 不触碰烙铁头：烙铁温度 300-400°C，接触会严重烫伤
4. 烙铁放回架：不用时立即放回烙铁架，不要放在桌上
5. 小心焊锡丝：焊锡可能含有铅，操作后洗手
6. 远离易燃物：烙铁、焊锡、助焊剂远离纸张、布料
7. 使用焊锡垫：在切割垫或焊锡垫上操作，保护桌面
8. 正确拔插：抓住插头拔插，不要拉扯电源线
9. 清洁烙铁：使用湿海绵或铜丝球清洁烙铁头
10. 收工断电：离开前确认烙铁已断电冷却
```

**认识焊接工具**：

```
电烙铁 Soldering Iron：
- 温度：300-400°C
- 功率：30-60W（推荐可调温焊台）
- 烙铁头形状：尖头（精密焊接）、刀头（大面积）

焊锡丝 Solder Wire：
- 直径：0.6mm - 1.0mm（推荐 0.8mm）
- 含铅 vs 无铅：含铅焊锡更容易操作（熔点低），但注意洗手
- 含有松香芯（助焊剂），焊接时不需要额外加助焊剂

助焊剂 Solder Flux：
- 作用：去除氧化层，帮助焊锡流动
- 使用：在焊接铜线前涂抹少量助焊剂
- 清洁：焊接后用酒精棉签清洁残留

吸锡器 Desoldering Pump：
- 用于拆除焊点
- 加热焊锡至熔化，按下吸锡器按钮吸取
```

### 2. 铜线弯制技巧 (1h)

**铜线材料认知**：

```
铜线规格选择：

1.0mm 直径铜线（主线）：
  - 用于主体结构框架
  - 较硬，弯制后不易变形
  - 需要较大的力弯制

0.7mm / 0.8mm 直径铜线（副线）：
  - 用于细节装饰和连接
  - 较软，容易弯制精细形状
  - 用于太阳能板支架、天线等

黄铜管 3mm 外径（结构件）：
  - 用于立柱和支撑
  - 可用铜线穿插连接
  - 切割使用小型割管器或锯条
```

**弯制基本技巧**：

```
基本弯制手法：

1. 圆形弯制：
   - 使用圆棒（如铁钉、钻头）作为模具
   - 将铜线绕模具一圈
   - 取下后微调形状

2. 直角弯制：
   - 用尖嘴钳夹住弯折点
   - 用力弯曲至 90°
   - 用平面校准角度

3. 螺旋弯制：
   - 将铜线沿圆柱体螺旋缠绕
   - 保持间距均匀
   - 取下后可作为装饰

4. 波浪形弯制：
   - 用尖嘴钳交替上下弯折
   - 保持一致的波浪幅度
```

**练习任务**：

1. 用 1.0mm 铜线弯制一个直径 5cm 的圆环
2. 用 0.7mm 铜线弯制一个等边三角形（边长 3cm）
3. 用铜线弯制一个简单的"卫星天线"形状（一根直线 + 顶部小球）

### 3. 黄铜管切割 (1h)

**切割工具和方法**：

```
黄铜管切割：

工具：
  - 迷你割管器（推荐，切口平整）
  - 小型钢锯（备选，需要锉刀修整）
  - 锉刀（去除毛刺）

切割步骤：
  1. 测量需要的长度（用尺子标记）
  2. 将割管器卡在标记位置
  3. 旋转割管器一圈
  4. 稍微拧紧，再旋转一圈
  5. 重复直到管子断开
  6. 用锉刀去除切口毛刺

安全注意：
  - 切割后的管口很锋利，小心划伤
  - 使用夹具固定管子，不要用手直接拿
  - 戴手套操作
```

### 4. 基础焊接练习 (2h)

**焊接铜线的五步法**：

```
焊接五步法：

1. 加热：烙铁头同时接触铜线和焊盘/另一根铜线
   ┌───┐
   │烙铁│──▶ 接触点
   └───┘
   时间：约 1-2 秒

2. 送锡：将焊锡丝送到接触点（不是烙铁上！）
   ┌───┐        ╱ 焊锡丝
   │烙铁│──▶ 接触点
   └───┘
   时间：约 1 秒

3. 观察：焊锡应自然流动，形成光滑的圆锥形
   合格焊点：     不合格焊点：
     ╱╲            ╱╲
    ╱  ╲          ╱球╲  (焊锡过多)
   ╱____╲        ╱____╲
   光滑圆润        或焊锡过少/虚焊

4. 撤锡：先移开焊锡丝
   时间：立即

5. 撤烙铁：再移开烙铁
   时间：等焊锡流动均匀后（约 0.5 秒）
   不要移动被焊件！等焊锡凝固（约 2 秒）
```

**焊接练习步骤**：

```cpp
/*

  - 两根铜线对齐

练习 2：铜线 T 形焊
       │
  ═════╪═════
       │
  - 一根铜线横放，另一根竖放在中间

       │

  - 在交叉点焊接

练习 4：铜线与黄铜管焊接
  ──[管]──

  - 在接口处焊接

练习 5：多点焊接
```

**常见焊接问题**：

| 问题 Problem | 原因 Cause | 解决方法 Solution |
|-------------|-----------|-----------------|
| 虚焊（焊点不牢固） | 加热不充分 | 延长加热时间，加助焊剂 |
| 焊锡不流动 | 铜线氧化 | 用砂纸打磨铜线表面，加助焊剂 |
| 焊锡过多 | 送锡太多 | 用吸锡器吸走多余焊锡 |
| 焊锡过少 | 送锡不够 | 重新加热补焊 |
| 焊点粗糙 | 移动过早 | 等焊锡完全凝固后再松手 |
| 烙铁不粘锡 | 烙铁头氧化 | 用湿海绵清洁，上锡保护 |
