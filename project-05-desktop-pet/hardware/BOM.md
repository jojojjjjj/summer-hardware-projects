# 物料清单 | Bill of Materials (BOM)

## 项目说明 | Project Notes

桌宠项目以**软件开发为核心**，硬件部分为**可选的拓展方向**。学员可以选择纯软件版本（0元），也可以选择添加硬件互动模块（约43元）。

The desktop pet project focuses on **software development**. Hardware is an **optional extension**. Students can choose the software-only version (free) or add hardware interaction modules (~43 RMB).

## 购买渠道 | Purchase Channels

| 渠道 | 特点 | 推荐指数 |
|------|------|----------|
| 淘宝 | 品种齐全，价格适中 | 首选 |
| 拼多多 | 价格最低，品质参差 | 推荐（注意选有销量的店铺） |
| 京东 | 品质有保障，价格稍高 | 可选 |
| 立创商城 | 电子元器件专业平台 | 推荐（元器件品质好） |

### 购买提示
1. 搜索时加上"模块"二字（如"DHT11 模块"），模块比裸传感器更方便使用
2. 选择有焊接好排针的模块，省去焊接步骤
3. 优先选择有销量和好评的店铺
4. 一次性购买所有部件，节省运费

## 替代方案 | Alternative Parts

如果买不到指定部件，以下为兼容替代：

| 原部件 | 替代方案 | 注意事项 |
|--------|----------|----------|
| Arduino Nano | Arduino Uno | 功能相同，体积更大 |
| Arduino Nano | ESP32/ESP8266 | 支持WiFi，但编程方式不同 |
| DHT11 | DHT22 | 精度更高，价格稍贵 |
| 面包板 | 洞洞板 | 需要焊接，不够灵活 |
