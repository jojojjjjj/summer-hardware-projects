# 课程总览 | Curriculum Overview

## 项目名称 | Project Name
桌宠 "萌桌面" | Desktop Pet "PetDesk"

## 项目定位 | Project Positioning
本项目是一个面向高中生的暑期实践课程，为期12天（约2周半），每天6-8小时。学员将从零开始，使用Python和PyQt5框架开发一个可以在电脑桌面上显示、行走、互动的虚拟宠物程序。

This is a 12-day summer practicum for high school students (6-8 hours per day). Students will build a virtual desktop pet from scratch using Python and PyQt5.

## 学习目标 | Learning Objectives

### 知识目标 | Knowledge Objectives
1. 掌握Python GUI编程基础（PyQt5框架）
2. 理解帧动画（Sprite Animation）的工作原理
3. 掌握有限状态机（FSM）设计模式
4. 理解事件驱动编程模型
5. 了解API调用和异步编程基础

### 技能目标 | Skill Objectives
1. 能够独立设计和实现一个完整的桌面应用程序
2. 能够使用状态机模式管理复杂的行为逻辑
3. 能够制作或编辑像素画帧动画素材
4. 能够编写清晰的技术文档和代码注释
5. 能够进行项目演示和技术表达

### 素养目标 | Competency Objectives
1. 培养"分解问题 -> 逐步解决"的工程思维
2. 培养独立调试和排错的能力
3. 培养项目管理和进度规划的意识
4. 培养技术文档撰写和表达能力

## 课程结构 | Course Structure

### 第一阶段：基础搭建（Day 1-4）
**主题：Python GUI基础与动画入门**

在这一阶段，学员将搭建开发环境，学习PyQt5基础，创建透明窗口，实现帧动画播放，掌握鼠标事件处理。完成本阶段后，学员将拥有一个能在透明窗口中播放动画的程序。

Students set up their development environment, learn PyQt5 basics, create transparent windows, implement sprite animation, and handle mouse events. By the end of this phase, students will have a program that plays animations in a transparent window.

### 第二阶段：核心功能（Day 5-8）
**主题：宠物行为与状态系统**

这一阶段是项目的核心，学员将学习状态机设计模式，实现宠物的多种行为（行走、睡觉、进食等），添加气泡对话框和音效系统。完成本阶段后，桌宠将具备完整的"生命感"。

This is the core phase. Students learn state machine design, implement pet behaviors (walking, sleeping, eating), add bubble dialogs and sound effects. After this phase, the desktop pet will have a complete "lifelike" feel.

### 第三阶段：拓展与展示（Day 9-12）
**主题：功能拓展与项目展示**

最后阶段涵盖拖拽交互、AI对话功能（可选）、系统打包发布和最终项目展示。学员将把所有功能整合为一个完整的产品，并向同学和老师展示成果。

The final phase covers drag interaction, AI dialogue (optional), system packaging, and the final demo. Students integrate all features into a complete product and present their work.

## 教学方法 | Teaching Methods

1. **项目驱动**：所有知识点都围绕桌宠项目展开，学以致用
2. **分步引导**：每个任务都提供详细的步骤说明，降低认知负荷
3. **即时反馈**：每完成一个任务都能立即看到桌宠的变化
4. **同伴学习**：鼓励学员之间互相帮助、代码审查
5. **自主探索**：每日作业鼓励个性化定制和创新尝试
