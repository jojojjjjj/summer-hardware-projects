## 上午 (3h) | Morning Session

### 1. 外壳设计原则 (1h)

**设计要求**：
- 保护电路板不受损坏
- 露出 OLED 屏幕窗口
- 留出 USB 接口和可能的按钮
- 美观大方

**设计工具**：
| 工具 | 难度 | 输出 | 适用 |
|------|------|------|------|
| Tinkercad | 极低 | STL (3D打印) | 零基础首选 |
| Fusion 360 | 中等 | STL | 有一定设计需求 |
| SolidWorks | 高 | STL | 专业设计 |
| 纸板 + 美工刀 | 极低 | 实物 | 最快速方案 |
| 亚克力板 | 低 | 实物 | 透明外壳 |

### 2. Tinkercad 快速入门 (1h)

**Tinkercad** (https://www.tinkercad.com/) 是一款免费的在线 3D 设计工具。

**基本操作**：
1. 注册账号 → 创建新设计
2. 从右侧拖拽基本形状（立方体、圆柱体等）
3. 组合形状 → 挖空（Hole）→ 合并
4. 导出 STL 文件

**外壳设计步骤**：
1. 测量实际尺寸（ESP-01S + 烧录器 + OLED）
2. 设计底座：略大于电路的方形盒子
3. 开窗口：前面板留出 OLED 屏幕大小的孔
4. 开接口：侧面留出 Micro-USB 接口
5. 添加固定柱：内部加支撑结构

### 3. 替代方案 (1h)

**如果无法 3D 打印**：
- **纸盒方案**：用硬纸板裁剪粘贴，成本 0 元
- **亚克力方案**：购买 2mm 亚克力板，裁切后用胶水粘合
- **裸板方案**：面包板直接放在桌面上，加一个亚克力支架

## 下午 (4h) | Afternoon Session

### 4. 整机组装 (1.5h)

**组装步骤**：
1. 确认所有功能正常（组装前最后测试一次）
2. 固定 ESP-01S + 烧录器（可用热熔胶或双面胶）
3. 固定 OLED 屏幕，对准外壳窗口
4. 整理杜邦线（可用扎带）
5. 合上外壳，测试是否影响散热和信号

**组装注意事项**：
- OLED 排线脆弱，不要用力弯折
- ESP8266 WiFi 天线区域不要被金属遮挡
- 保留散热空间，不要完全密封
- 先不封死，测试通过后再固定

### 5. 最终系统测试 (1.5h)

**完整测试清单**：

```
[ ] 上电启动 → 启动画面正常
[ ] WiFi 连接 → 自动连接成功
[ ] 时间显示 → 正确（与手机对比）
[ ] 天气数据 → 正确获取并显示
[ ] 天气图标 → 与实际天气对应
[ ] PM2.5/B站粉丝 → 数据正常
[ ] 配网功能 → 可重新配网
[ ] 连续运行 2 小时 → 无死机、无重启
[ ] 断电重启 → 自动恢复工作
[ ] 外壳装配 → 美观、散热正常
```
