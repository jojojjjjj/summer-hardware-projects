# 前置知识 | Prerequisites

## 必备基础 | Required Basics

### C 语言基础
- 变量与数据类型（int, float, char, String）
- 条件判断（if/else if/else）
- 循环（for, while）
- 函数定义与调用
- 数组基础

### 计算机基础
- 了解什么是编程（写代码让硬件做事）
- 了解什么是文件（.ino, .h, .cpp 文件类型）
- 会使用浏览器和搜索引擎查找资料

### 硬件基础
- 知道什么是电压（3.3V vs 5V）
- 知道什么是 GND（地线）
- 不需要焊接经验（本项目使用面包板 + 杜邦线）

## 名词速查 | Quick Glossary

| 名词 | 解释 |
|------|------|
| **MCU** | 微控制器（单片机），一颗小小的"电脑芯片" |
| **ESP8266** | 一款带 WiFi 功能的便宜 MCU（8块钱） |
| **GPIO** | 通用输入输出引脚，MCU 的"手和眼睛" |
| **I2C** | 一种两线制通信协议（SCL时钟 + SDA数据） |
| **OLED** | 有机发光二极管显示屏，自发光，不需要背光 |
| **SSD1306** | 控制 OLED 屏幕的驱动芯片型号 |
| **WiFi** | 无线局域网，MCU 通过它连上互联网 |
| **API** | 应用程序接口，一个可以"打电话索取数据"的网址 |
| **JSON** | 一种数据格式，类似"标签:值"的结构化文本 |
| **NTP** | 网络时间协议，从网络服务器获取精确时间 |
| **HTTPS** | 加密版的 HTTP，数据在传输过程中是安全的 |
| **UDP** | 用户数据报协议，一种"发出去就不管了"的网络通信方式 |
