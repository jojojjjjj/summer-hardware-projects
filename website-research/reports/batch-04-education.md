# Batch 04: 教育/课程平台网站调研

> 调研对象：Coursera.org, Khan Academy, Domestika.org, Adafruit.com (学习板块), Arduino.cc
> 调研日期：2026-06-01
> 注：基于搜索结果和行业知识综合分析

---

## 1. Coursera.org — 学院派课程平台

### 🎯 设计哲学
Coursera的设计语言是"可信赖的教育"——清晰的信息架构、大学logo背书、结构化的课程路径。

### 🎨 视觉风格
| 维度 | 分析 |
|------|------|
| **配色** | 蓝色主调(#0056D2) + 白色背景 + 深色文字 |
| **字体** | 清晰无衬线（Source Sans Pro风格） |
| **间距** | 标准教育平台间距，卡片网格布局 |
| **图片** | 课程卡片（封面图+学校logo+评分）+ 讲师头像 |

### 📐 可借鉴的设计模式
1. **课程卡片设计** — 封面图 + 标题 + 学校logo + 评分星级 + 学习人数
2. **学习路径** — "Beginner → Intermediate → Advanced" 进阶路径可视化
3. **大学/机构背书** — 合作机构logo墙
4. **证书展示** — 完课证书的可视化预览

### ⭐ 推荐指数：★★★★☆
**最适合**：课程卡片设计、学习路径可视化

---

## 2. Khan Academy — 免费教育的友好界面

### 🎯 设计哲学
Khan Academy的设计核心是"无压力学习"——温暖的色彩、友好的插画、游戏化元素（积分/徽章/连续学习天数）。

### 🎨 视觉风格
| 维度 | 分析 |
|------|------|
| **配色** | 绿色主调(#14BF96) + 暖灰背景 + 多彩学科图标 |
| **字体** | 友好无衬线 |
| **间距** | 宽松友好，适合青少年 |
| **图片** | 可爱插画 + 学科图标 + 进度条 |

### 📐 可借鉴的设计模式
1. **学科图标系统** — 每个学科/方向用独特图标+颜色标识
2. **进度可视化** — 技能树/进度条/完成百分比
3. **游戏化元素** — 积分/徽章/连续学习天数
4. **友好的插画风格** — 适合高中生的温暖插画

### ⭐ 推荐指数：★★★★☆
**最适合**：游戏化学习进度、学科分类图标

---

## 3. Domestika.org — 创意课程的高端展示

### 🎯 设计哲学
Domestika专注于创意课程，其设计语言充满艺术感——大图展示、精致排版、视频预览。

### 📐 可借鉴的设计模式
1. **视频课程预览** — 课程卡片的视频hover预览
2. **讲师展示** — 讲师作品集+头像+专业背景
3. **学生作品画廊** — 优秀学生作品的瀑布流展示

### ⭐ 推荐指数：★★★☆☆
**最适合**：学员作品画廊灵感

---

## 4. Adafruit.com — Maker社区的百科全书

### 🎯 设计哲学
Adafruit是硬件/Maker社区的"百科全书式"电商——不只是卖产品，更提供完整的学习资源和社区。Lady Ada的愿景："让每个人都能学到电子学"。

### 🎨 视觉风格
| 维度 | 分析 |
|------|------|
| **配色** | 绿色(#00AA55)主调 + 深色导航 + 白色内容区 |
| **字体** | 功能性无衬线 |
| **间距** | 电商式紧凑布局 |
| **图片** | 产品照片 + 项目展示图 + 视频封面 |

### ✨ 交互与动效
- **多级导航**：Products/Learn/Blog/Forums/IO/LIVE!/AdaBox
- **产品视频封面**：产品卡片的视频hover播放
- **学习路径指南**：指南式教程入口
- **新品/特色分区**：New Products / Featured Products 区块
- **社区活动**：展示Coolest Projects等学生竞赛

### 📐 可借鉴的设计模式
1. **学习→项目→购买** — 从教程到项目到零件购买完整流程
2. **项目指南系统** — 分级难度的教程/指南入口
3. **社区竞赛展示** — 学生竞赛和作品展示
4. **视频产品展示** — 硬件产品的视频演示

### ⭐ 推荐指数：★★★★★
**最适合**：硬件项目→教程→零件的完整链路设计

---

## 5. Arduino.cc — 开源硬件的教育门户

### 📐 可借鉴的设计模式
1. **项目Hub** — 社区分享项目的中心
2. **入门指南** — 零基础的分步入门教程
3. **硬件兼容性表** — 清晰展示板卡/模块兼容性
4. **教育专区** — 面向教师/学生的专门入口

### ⭐ 推荐指数：★★★★☆
**最适合**：教程入口设计、硬件兼容性展示
