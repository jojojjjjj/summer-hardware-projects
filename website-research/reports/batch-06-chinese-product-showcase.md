# Batch 06: 中国/亚洲科技教育网站 + 产品落地页 + 展示型网站

> 调研对象：Bilibili, OSHWhub(立创开源), Espressif(乐鑫), Framework Laptop, MNT Reform, Payload CMS
> 调研日期：2026-06-01

---

## 1. Bilibili.com — 弹幕视频社区

### 🎯 设计语言特点
- **中文排版优化**：16px基础字号，行高1.75，中文友好的字体栈
- **视频封面网格**：响应式网格布局展示视频封面
- **标签/分区系统**：科技/游戏/知识等分区导航
- **弹幕文化元素**：弹幕预览、互动按钮

### 📐 可借鉴的设计模式
1. **视频封面网格** — 项目视频展示的网格布局
2. **分区导航** — 按难度/方向分类的项目筛选
3. **弹幕/评论展示** — 学员评价的弹幕式展示
4. **UP主（作者）卡片** — 项目作者/讲师的信息卡片

## 2. OSHWhub.com (立创开源) — 硬件开源平台

### 🎯 设计语言特点
- **工程文件预览**：PCB/原理图的在线预览
- **BOM清单展示**：物料清单的结构化表格
- **项目分类标签**：MCU类型/应用场景/难度等级
- **开源协议标识**：MIT/GPL等协议清晰标注

### 📐 可借鉴的设计模式
1. **工程文件预览** — 项目硬件设计的在线预览
2. **BOM清单表格** — 物料清单的结构化展示
3. **难度标签系统** — ⭐到⭐⭐⭐⭐⭐的难度评级
4. **开源信息** — 开源仓库链接 + 协议类型 + Star数

### ⭐ 推荐指数：★★★★★
**最适合**：硬件项目展示、物料清单、开源信息展示

### 🎯 设计语言特点

- **技术文档入口**：清晰的文档/API/工具下载入口

### 📐 可借鉴的设计模式

2. **文档层次** — Getting Started → API Reference → Examples
3. **双语入口** — 中文/英文切换

## 🎯 中文本地化设计建议

### 字体选择
```css
/* 推荐字体栈 */
--font-sans: "Inter", "Noto Sans SC", "PingFang SC", "Microsoft YaHei", sans-serif;
--font-mono: "JetBrains Mono", "Fira Code", "Source Code Pro", monospace;
```

### 排版参数
| 参数 | 中文建议值 | 英文建议值 |
|------|-----------|-----------|
| 基础字号 | 16-18px | 16px |
| 行高 | 1.75-2.0 | 1.5-1.6 |
| 段落间距 | 1.5em | 1em |
| 字间距 | 0.05em | normal |
| 最大行宽 | 680px | 720px |

### 双语设计模式
1. **Tab切换**：中文/英文 Tab 切换显示
2. **并排显示**：左侧中文，右侧英文（参考本项目README格式）
3. **段落交替**：每段先中文后英文（当前项目的模式）
4. **语言检测**：根据浏览器语言自动切换
