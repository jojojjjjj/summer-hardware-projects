# 组装步骤 | Assembly Steps

本文档提供"ThermalEyes"双目手机热成像仪的详细硬件组装指导，从 PCB 打样到最终调试的完整流程。

> This document provides detailed hardware assembly instructions for the "ThermalEyes" binocular thermal imager, covering the complete process from PCB fabrication to final testing.

## 第一步：PCB 打样 | Step 1: PCB Fabrication

### 使用 OSHWhub 开源设计直接打样

1. **打开工程页面**：
   访问 https://oshwhub.com/colourfate/binocular_thermal_imager

2. **克隆工程**：
   点击页面上的"克隆工程"按钮，工程会复制到你的账号下

3. **在嘉立创EDA中打开**：
   登录嘉立创EDA（在线版或桌面版均可），找到克隆的工程

4. **检查设计**：
   - 打开 PCB 编辑器，检查走线是否完整
   - 确认元件封装是否正确
   - 检查板子尺寸（约 50x30mm）

5. **下单打样**：
   - 在嘉立创EDA中点击"一键下单"或访问 https://www.jlc.com/
   - 参数选择：
     - 板层数：2层
     - 板厚：1.6mm
     - 铜厚：1oz (35um)
     - 表面处理：无铅喷锡（HASL）或 沉金（ENIG），建议沉金对贴片焊接更友好
     - 阻焊颜色：绿色（最便宜）或自选
     - 最小 5 片起订
   - 价格约 ¥5（基础配置）

6. **等待收货**：
   嘉立创打样通常 2-3 天生产，快递 1-3 天

### 如果要自己设计 PCB（进阶）

如果你有 PCB 设计经验，可以自己绘制。以下是最小系统板的设计要点：

1. **元件布局**：MCU 居中，USB Type-C 在一端，传感器接口在另一端
2. **去耦电容**：紧贴 MCU 的 VDD 引脚放置
3. **USB 走线**：D+/D- 平行等长走线，差分阻抗 90 欧姆
4. **I2C 走线**：尽量短且直
5. **地平面**：底层大面积铺地，减少噪声
6. **丝印标注**：标注每个接口的名称，方便调试

## 第二步：焊接元件 | Step 2: Soldering Components

### 焊接顺序原则 | Soldering Order

**从矮到高、从小到大**：先焊贴片元件，再焊插件元件。

> **Principle**: Solder from low to high, small to large. Start with SMD components, then through-hole.

### 阶段 2.1：电源电路 | Power Circuit

**焊接顺序**：USB Type-C → LDO → 电源滤波电容

1. **焊接 USB Type-C 母座**（如果使用沉板式）：
   - 先对齐母座到 PCB 焊盘上
   - 先焊一个固定脚定位
   - 检查位置是否端正
   - 焊接其余引脚
   - 用万用表检查相邻引脚是否短路

2. **焊接 AMS1117-3.3 (LDO)**：
   - 确认封装方向：SOT-223 的 tab 面（大金属面）是输出 (VOUT)
   - 涂少量助焊剂在焊盘上
   - 先焊一个引脚定位
   - 焊接其余引脚
   - 检查焊点是否光亮

3. **焊接电源电容**（10uF x2）：
   - 输入电容：LDO 输入端 (VIN 和 GND 之间)
   - 输出电容：LDO 输出端 (VOUT 和 GND 之间)

**阶段测试：电源**
```
1. 用 USB 线连接 PCB 和电脑或 5V 充电器
2. 万用表测量 LDO 输出 (VOUT)：应为 3.3V ± 0.1V
3. 测量 PCB 上的 3.3V 网络：应为 3.3V
4. 拔掉 USB，测量 3.3V 和 GND 之间：应有一定电阻（kΩ级别），不应短路（0欧）
5. 如果无输出：检查 LDO 方向、输入是否正常
6. 如果输出不是 3.3V：检查 LDO 型号是否正确（不是 AMS1117-5.0）
```

### 阶段 2.2：MCU 最小系统 | MCU Minimum System

**焊接顺序**：MCU → 去耦电容 → 晶振 → 复位电路 → BOOT电阻

1. **焊接 STM32F411CEU6 (LQFP48)**：
   这是焊接难度最高的部分，请仔细操作。

   **步骤详解**：
   a. 在 PCB 的 MCU 焊盘上涂薄薄一层助焊剂
   b. 将芯片放在焊盘上，**对齐第 1 脚**（芯片上的小圆点对应 PCB 丝印的 Pin 1 标记）
   c. 先用烙铁焊对角的两个引脚固定位置
   d. 检查所有引脚是否对齐焊盘
   e. 在芯片引脚一侧涂助焊剂
   f. 烙铁头上带少量焊锡，沿引脚方向拖焊
   g. 用放大镜检查是否有连锡（桥接）
   h. 如有连锡，涂助焊剂后用烙铁重新拖一下，或用吸锡带吸走多余焊锡
   i. 重复 e-h 处理其余三边

   **新手贴片焊接技巧**：
   - 烙铁温度设置 320-350°C
   - 烙铁头保持干净，定期在湿海绵或铜丝球上擦拭
   - 助焊剂是关键——焊点不亮就多涂一点
   - 不要着急，每个引脚都要仔细检查
   - 如果一个引脚焊不好，涂助焊剂重来
   - 拖焊时烙铁速度不要太快，让焊锡自然流动

2. **焊接去耦电容**（100nF x4）：
   - 放在 MCU 每对 VDD/VSS 引脚旁边
   - 0402 封装很小，用镊子夹住放到涂了助焊剂的焊盘上
   - 先焊一端固定，再焊另一端

3. **焊接晶振电路**：
   - 8MHz 晶振放在 PH0/PH1 附近
   - 两个 20pF 电容分别从晶振两端接 GND

4. **焊接复位电路**：
   - NRST 引脚接 10K 电阻到 3.3V（上拉）
   - 复位按键接 NRST 到 GND

5. **焊接 BOOT0 电阻**：
   - 10K 电阻从 BOOT0 (Pin 44) 接到 GND

**阶段测试：MCU 供电与 SWD**
```
1. 上电 (USB 连接)
2. 万用表测量 MCU 的 VDD 引脚：应为 3.3V
3. 测量 NRST 引脚：应为 3.3V（高电平，正常运行）
4. 连接 ST-Link：
   - ST-Link SWDIO → MCU PA13
   - ST-Link SWCLK → MCU PA14
   - ST-Link GND → PCB GND
   - ST-Link 3.3V → 不要连接（板子已有 3.3V）
5. 打开 STM32CubeIDE，配置 ST-Link
6. 尝试读取 MCU 芯片 ID
7. 如果能读取到芯片 ID，说明 MCU 焊接正确
8. 如果无法连接：
   - 检查 SWDIO/SWCLK 是否正确连接
   - 检查 MCU 方向是否正确
   - 检查 NRST 和 BOOT0 电阻
   - 检查供电是否正常
```

### 阶段 2.3：MLX90640 热成像接口 | Thermal Sensor Interface

1. **焊接 I2C 上拉电阻**（4.7K x2）：
   - PB6 (SCL) → 4.7K → 3.3V
   - PB7 (SDA) → 4.7K → 3.3V
   - 如果 MLX90640 模块已自带，可以跳过

2. **焊接 MLX90640 排针/排母**（如果使用模块方式）：
   - 在 PCB 上焊接 4pin 排母
   - MLX90640 模块插在排母上
   - 或者直接将 MLX90640 模块焊在 PCB 上

3. **连接 MLX90640 模块**：
   - VDD → 3.3V（注意不是 5V）
   - GND → GND
   - SCL → PB6
   - SDA → PB7

**阶段测试：MLX90640**
```
1. 上电
2. 用万用表测量 MLX90640 模块 VDD：应为 3.3V
3. 测量 SCL 和 SDA 线：都应约为 3.3V（上拉状态）
4. 通过 ST-Link 烧录一个简单的 I2C 扫描程序：
   - 扫描 I2C1 总线
   - 应在地址 0x33 处发现 MLX90640
5. 如果扫描不到：
   - 检查接线是否正确
   - 检查上拉电阻
   - 尝试降低 I2C 速度到 100kHz
```

### 阶段 2.4：OV2640 摄像头接口 | Camera Interface

1. **焊接 24pin FPC 座子**：
   - FPC 座子是细间距的贴片元件，需要仔细对齐
   - 先涂助焊剂在焊盘上
   - 将座子对齐放到焊盘上
   - 先焊两端的固定焊点
   - 检查位置后焊接所有引脚

2. **焊接 SCCB 上拉电阻**（4.7K x2）：
   - PA8 (SCCB_SCL) → 4.7K → 3.3V
   - PA9 (SCCB_SDA) → 4.7K → 3.3V

3. **插入 OV2640 FPC 排线**：
   - 打开 FPC 座子的翻盖
   - 将排线金手指面朝下（朝向座子触点）
   - 轻轻推入到底
   - 压紧翻盖锁定

   **注意**：FPC 排线非常脆弱，请勿反复弯折。插入方向错误会导致摄像头无法工作。

**阶段测试：OV2640**
```
1. 上电
2. 测量 OV2640 模块 VDD：应为 3.3V
3. 通过 ST-Link 烧录 SCCB 读写测试程序：
   - 尝试读取 OV2640 的芯片 ID 寄存器
   - 应读取到 0x2642 (OV2640) 或 0x5640 (OV5640)
4. 如果读不到 ID：
   - 检查 FPC 排线方向和插入是否到位
   - 检查 SCCB 上拉电阻
   - 尝试降低 SCCB 时钟频率
```

### 阶段 2.5：USB 接口与外围 | USB Interface & Peripherals

1. **焊接 USB 阻抗匹配电阻**（22R x2）：
   - PA12 (USB_DP) → 22R → USB Type-C D+
   - PA11 (USB_DM) → 22R → USB Type-C D-

2. **焊接 USB D+ 上拉电阻**（1.5K）：
   - PA12 (USB_DP) → 1.5K → 3.3V
   - 这个电阻告诉手机这是 USB 全速设备

3. **焊接 Type-C CC 下拉电阻**（5.1K x2）：
   - CC1 → 5.1K → GND
   - CC2 → 5.1K → GND

4. **焊接 LED 指示灯**：
   - 红色 LED：PC13 → 1K 电阻 → LED → GND
   - 绿色 LED：PC14 → 1K 电阻 → LED → GND

**阶段测试：USB**
```
1. 上电
2. 用 USB-C 线连接 PCB 和 Android 手机
3. 手机应弹出 USB 设备连接通知
4. 如果手机不识别：
   - 检查 D+ 上拉电阻（1.5K）是否正确
   - 检查 CC 下拉电阻（5.1K x2）是否安装
   - 检查 USB 线是否支持数据传输（不是纯充电线）
   - 尝试连接电脑查看设备管理器是否出现新设备
```

## 第三步：固件烧录 | Step 3: Firmware Flashing

### 安装 STM32CubeIDE

1. 下载 STM32CubeIDE：https://www.st.com/en/development-tools/stm32cubeide.html
2. 安装时选择默认选项
3. 安装 ST-Link 驱动

### 烧录固件

1. **克隆固件仓库**：
   ```bash
   git clone https://github.com/colourfate/thermal_bridge.git
   ```

2. **打开工程**：
   - 启动 STM32CubeIDE
   - File → Open Projects from File System → 选择 thermal_bridge 文件夹

3. **编译工程**：
   - Project → Build All
   - 确认无编译错误

4. **连接 ST-Link**：
   - ST-Link 连接到 PCB 的 SWD 调试接口
   - 确认 ST-Link 指示灯亮起

5. **烧录**：
   - Run → Debug (F11) 或 Run → Flash
   - 等待烧录完成
   - MCU 将自动复位并开始运行

### 首次测试

烧录固件后，进行以下测试：

1. **LED 测试**：
   - 红色 LED 应常亮（表示有电）
   - 绿色 LED 可能闪烁（表示初始化或数据传输）

2. **USB 连接测试**：
   - 用 USB-C 线连接 PCB 和手机
   - 手机 APP 应能检测到 USB 设备
   - 授权 USB 权限后开始数据传输

3. **MLX90640 数据测试**：
   - 用手指靠近 MLX90640 镜头
   - 手机 APP 上应能看到温度变化

4. **OV2640 图像测试**：
   - 手机 APP 上应能看到可见光画面
   - 画面应清晰，无花屏或条纹

## 第四步：最终组装 | Step 4: Final Assembly

### 方案 A：裸板使用（最简单）

直接使用焊接好的 PCB，无需外壳。注意以下几点：

1. MLX90640 镜头避免触摸和刮擦
2. OV2640 镜头避免触摸
3. 使用时避免短路 PCB 背面
4. 可以在 PCB 背面贴一层绝缘胶带

### 方案 B：3D 打印外壳（推荐）

OSHWhub 项目提供了 3D 打印外壳的 STL 文件：

1. **下载 STL 文件**：
   从 https://oshwhub.com/colourfate/binocular_thermal_imager 的"附件"中下载 `3d_printed_casing.zip`

2. **3D 打印**：
   - 使用 PLA 或 PETG 材料
   - 层高 0.2mm
   - 无需支撑（根据具体 STL 设计）
   - 可以使用学校/图书馆的 3D 打印机
   - 也可以找淘宝 3D 打印代工，约 ¥10-20

3. **组装外壳**：
   - 将 PCB 放入外壳底座
   - 对准 USB Type-C 接口和镜头开孔
   - 用螺丝或卡扣固定
   - 盖上外壳顶部

### 方案 C：纸盒外壳（应急方案）

如果没有 3D 打印条件：

1. 找一个合适大小的硬纸盒
2. 在对应位置开孔：
   - USB Type-C 接口孔
   - MLX90640 镜头孔（约 10mm 直径）
   - OV2640 镜头孔（约 8mm 直径）
3. 用双面胶或热熔胶固定 PCB
4. 可以用黑色电工胶布包裹外观

## 组装检查清单 | Assembly Checklist

完成所有组装后，逐项确认：

```
焊接质量：
□ 所有焊点光亮、无虚焊
□ 无焊锡桥接（连锡）
□ 助焊剂残留已用酒精清洗干净
□ PCB 正反面无明显短路

电源系统：
□ USB 供电正常
□ LDO 输出 3.3V 稳定
□ 各 VDD 引脚电压正确

MCU 系统：
□ STM32 芯片方向正确
□ 晶振起振（可用示波器观察 PH0 引脚应有 8MHz 正弦波）
□ ST-Link 能连接并读取芯片 ID
□ 固件烧录成功

传感器系统：
□ MLX90640 模块连接正确，I2C 通信正常
□ OV2640 摄像头排线正确插入，SCCB 通信正常
□ 两个传感器镜头无遮挡

USB 通信：
□ USB 连接线数据线（非纯充电线）
□ 手机能识别 USB 设备
□ APP 能正常接收数据

外观：
□ PCB 清洁
□ 外壳安装（如有）
□ 标签或标记
```

## 常见组装问题 | Common Assembly Issues

### Q1: MCU 焊接后 ST-Link 连不上

**可能原因**：
- MCU 方向焊反（第 1 脚位置错误）
- 引脚桥接（连锡）
- 供电异常
- SWD 引脚（PA13/PA14）被其他外设占用

**解决方法**：
1. 检查 MCU 第 1 脚位置（芯片上的小圆点应对应 PCB 丝印标记）
2. 用放大镜检查每个引脚是否有桥接
3. 测量 VDD 电压是否为 3.3V
4. 测量 NRST 电压是否为 3.3V（高电平）
5. 确认 BOOT0 为低电平

### Q2: MLX90640 扫描不到

**可能原因**：
- I2C 接线错误
- 上拉电阻缺失
- MLX90640 供电异常
- I2C 速度设置过快

**解决方法**：
1. 确认 SCL 和 SDA 未接反
2. 测量 SCL/SDA 线上的电压，空闲时应为 3.3V
3. 确认 MLX90640 VDD 为 3.3V
4. 降低 I2C 速度到标准模式 (100kHz) 重新尝试

### Q3: OV2640 画面全黑或花屏

**可能原因**：
- FPC 排线方向错误
- FPC 排线未插紧
- SCCB 通信异常
- PWDN 引脚处于高电平（断电状态）

**解决方法**：
1. 取出 FPC 排线，确认金手指方向正确
2. 重新插入排线，确保推到底
3. 检查 SCCB 上拉电阻
4. 确认 PWDN 引脚（如果有使用）为低电平

### Q4: USB 连接手机无反应

**可能原因**：
- D+ 上拉电阻未安装
- CC 下拉电阻未安装
- 使用了纯充电线（无数据功能）
- USB 线序错误

**解决方法**：
1. 确认 1.5K D+ 上拉电阻已安装
2. 确认 5.1K CC 下拉电阻 x2 已安装
3. 更换 USB 数据线
4. 尝试连接电脑，检查设备管理器

## 安全注意事项 | Safety Notes

1. **防静电**：触摸 PCB 和元件前，先用手摸一下金属物体（如桌子金属腿）释放静电。有条件的话佩戴防静电手环。
2. **烙铁安全**：电烙铁温度高达 350°C，切勿触摸烙铁头。不用时放回烙铁架。离开工位时断电。
3. **通风**：焊接时会产生烟尘（主要是助焊剂挥发），保持工作区域通风。
4. **MLX90640 保护**：红外传感器的镜头表面有镀膜，不要用手指触摸，不要用酒精或溶剂擦拭。如有灰尘用气吹清理。
5. **USB 供电**：本项目仅从 USB 取电，电流很小（约 200mA），安全。但不要同时连接 ST-Link 供电和 USB 供电，避免电源冲突。
6. **助焊剂清洗**：焊接完成后用无水乙醇清洗残留的助焊剂，避免长期腐蚀焊点。
