# 作业汇总 | Assignments Summary

本文档汇总了所有 15 天的课程作业。

> This document summarizes all assignments for the 15-day course.

---

## Day 1: 项目介绍 + 环境搭建

### 必做作业
1. 环境验证
   - 提交 STM32CubeIDE 编译成功的截图
   - 提交工程目录截图

2. 学习笔记
   - 创建 notes.md
   - 画出系统架构图
   - 写出三个疑问

### 推荐作业
- 克隆固件仓库，浏览目录结构

### 可选作业
- 查阅 MLX90640 数据手册的 Features 部分

---

## Day 2: STM32 基础与 GPIO

### 必做作业
1. LED 闪烁实验
   - 修改闪烁频率，观察效果
   - 提交代码截图和运行效果照片

2. 按键实验
   - 实现按键控制 LED
   - 提交代码和说明

3. 理解检查
   - 解释 GPIO、while(1) 循环、USER CODE 区域

### 可选作业
- 实现 LED 呼吸灯或多种闪烁模式

---

## Day 3: I2C 通信原理与实践

### 必做作业
1. I2C 扫描实验
   - 成功扫描到 MLX90640（0x33）
   - 截图串口输出

2. 原理理解
   - 画出 I2C 写操作时序图
   - 解释 I2C 的线、上拉电阻、主从设备

### 推荐作业
- 故意接错线，观察扫描结果（调试实践）

### 可选作业
- 添加第二个 I2C 设备，验证多设备共存

---

## Day 4: MLX90640 传感器原理

### 必做作业
1. 传感器信息读取
   - 读取 Chip ID 和控制寄存器
   - 提交串口输出截图

2. 数据手册笔记
   - 整理关键参数表
   - 列出 5 个常用寄存器

### 推荐作业
- 设置不同刷新率并验证

### 可选作业
- 思考题：为什么热像仪分辨率这么低？

---

## Day 5: MLX90640 基础驱动开发

### 必做作业
1. EEPROM 和帧数据读取
   - 成功读取校准参数和帧数据
   - 提交截图

2. 数据分析
   - 记录校准参数值和前 20 个原始值

### 推荐作业
- 将驱动代码整理到 mlx90640.c/h 文件

### 可选作业
- 思考题：为什么每个传感器需要单独的校准参数？

## 作业提交要求 | Submission Requirements

### 格式要求
- 代码提交到 Git 仓库
- 文档使用 Markdown 格式
- 截图使用 PNG 或 JPG 格式

### 命名规范
```
project-01-binocular-thermal-imager/
├── curriculum/       -- 课程文件
├── firmware/         -- STM32 固件代码
├── app/              -- Android APP 代码
├── hardware/         -- 硬件相关文件
├── docs/             -- 文档
│   ├── notes.md      -- 学习笔记
│   └── protocol.md   -- 通信协议文档
└── assignments/      -- 作业提交
    ├── day-01/
    ├── day-02/
    └── ...
```

### 提交时间
- 每天作业当天 23:59 前提交
- 延迟提交扣 10% 分数/天
