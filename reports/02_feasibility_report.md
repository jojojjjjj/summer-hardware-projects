## 二、详细评语 | Detailed Reviews（Top 6）

### 1. 智能音箱（Whisper + Edge TTS + 唤醒词）— 8.55 分

**核心优势：** 完整的语音交互链路（STT → LLM → TTS），技术栈全部基于 Python，与高中生编程入门完美契合。Edge TTS 零配置即可使用，Whisper tiny 可在树莓派上运行。唤醒词检测有现成开源实现。

**主要风险：** 树莓派 4 上 Whisper tiny 延迟约 2-3 秒；本地 LLM 推理需要 API 调用或量化模型。

**改编建议：** 分三个阶段——先实现文字对话，再加语音输入，最后加语音输出和唤醒词。可选用 RPi 5 降低延迟。

---

### 2. MediaPipe 手势互动装置 — 8.55 分

**核心优势：** Google 官方维护，Python API 极其友好，10 行代码即可实现手势检测。视觉效果震撼，非常适合最终展示。从手势控制音乐播放器到互动游戏都有丰富扩展空间。

**主要风险：** 需要较好的摄像头和计算性能；光照条件影响识别效果。

**改编建议：** 可用笔记本电脑摄像头替代树莓派，降低硬件成本。从手部关键点检测入手，逐步扩展到姿态、面部等。

---

### 3. ESPHome 室内环境监测站 — 8.15 分

**核心优势：** ESPHome 的 YAML 配置方式极大降低了编程门槛，非常适合零基础学员。同时采集温度、湿度、气压、PM2.5、CO₂ 等多维度数据，教学价值极高。可接入 Home Assistant 或 Grafana 可视化。

**主要风险：** ESPHome 对高级定制有一定限制；若需要 Python 编程教学，需切换到原生 ESP32 开发。

**改编建议：** 前半程用 ESPHome 快速搭建，后半程过渡到 MicroPython 自行实现传感器读取和数据上传，兼顾低门槛和编程深度。

---

### 4. ESP32-CAM 运动检测+通知 (pycameresp) — 7.65 分

**核心优势：** 成本极低（<!-- ¥150 以内 -->），结合 IoT 和 CV 双方向，实用性强（可做简易安防）。

**主要风险：** ESP32 内存有限，复杂 CV 算法无法运行；调试需要串口工具。

**改编建议：** 适合作为 IoT+CV 混合方向的轻量项目，运动检测由 ESP32 完成，复杂分析由 PC/RPi 完成。

---

### 5. Klimerko 空气质量监测 — 7.35 分

**核心优势：** 专为 DIY 设计，文档非常完整，社会意义强（空气质量关注）。

**主要风险：** 主要基于 C++，对零基础高中生有一定门槛；功能相对单一。

**改编建议：** 可用 MicroPython 重写核心逻辑，降低编程难度。

---

### 6. 桌宠机器人 — 7.05 分

**核心优势：** 对高中生吸引力最大，视觉冲击力强，涉及舵机控制、表情显示、语音交互等多模态技术。

**主要风险：** 需要 3D 打印外壳或手工制作；机械结构调试耗时；多技术栈整合难度高。成本可能超 <!-- ¥500 -->。

**改编建议：** 简化为"桌面对话伙伴"——用 RPi + 小屏幕 + 扬声器，跳过复杂机械结构。或用纸板 + 舵机做简易版。
