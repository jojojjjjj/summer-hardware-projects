ElectronBot

ElectronBot

稚晖君设计的迷你桌面机器人，灵感来源于 WALL-E 中的 EVE。拥有圆形 LCD 脸部、6 自由度舵机系统、手势传感器，通过 USB HS 连接 PC，配合 Unity 和 OpenCV/OpenPose 实现人体姿态跟随与表情交互。全栈项目：从 PCB 到上位机。

高级 15 天 ~¥500 STM32F405 + STM32F042×6
ElectronBot

STM32F405 + STM32F042×6

STM32 + USB HS + Unity + OpenPose

课程天数

15 天

预估成本

~¥500

  • QFN 焊接

软件

  • USB Device 开发

  • OpenCV/OpenPose

系统
