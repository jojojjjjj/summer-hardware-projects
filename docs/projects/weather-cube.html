天气魔方

Weather Cube

使用 ESP8266 驱动 0.96 寸 OLED 显示屏，通过 WiFi 获取实时天气数据，展示时间、日期、天气图标、体感温度、PM2.5 等信息。掌握嵌入式 C 编程、WiFi 联网、HTTP API、JSON 解析等核心 IoT 技能。
