Project 09

桌面卫星气象站

Desk Satellite

使用 ESP32-C3 驱动 TFT 彩屏，搭载 SHT31-D 高精度温湿度传感器，支持 WiFi NTP 时间同步、Type-C + 太阳能双充电，铜线+黄铜管手工弯制卫星造型外壳。

进阶 14 天 ~¥250 ESP32-C3
