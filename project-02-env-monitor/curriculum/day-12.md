### 演示模板 | Presentation Template

**开场(1分钟):**

```
"大家好,我是[姓名],今天演示的项目是'空气管家'环境监测站。

这个项目能够实时监测室内空气质量,包括温度、湿度、气压、
PM2.5等指标,并通过网络上传到云端进行可视化分析。

演示将包括以下几个部分:
1. 硬件展示
2. 功能演示
3. 技术细节
4. 项目总结"
```

**硬件展示(2分钟):**

```
"这是我们的原型机,主要包括:

硬件组件:
- ESP32微控制器(大脑)
- BME680环境传感器(温湿度/气压)
- PMS5003 PM2.5传感器
- OLED显示屏(本地显示)

设计亮点:
- 紧凑的外壳设计(100×80×50mm)
- 良好的气流设计
- 模块化组件布局"
```

**功能演示(4分钟):**

```
"现在演示系统功能:

第一步: 本地监测
大家可以看到OLED屏幕实时显示:
- 当前温度: 25.5°C
- 湿度: 60%
- PM2.5: 45 μg/m³

第二步: 数据上传
设备通过WiFi连接到MQTT Broker,
每60秒上传一次数据。

第三步: 远程监控
打开Grafana仪表盘,我们可以看到:
- 实时数据更新
- 历史趋势曲线
- AQI计算结果

演示完成!"
```

**技术总结(2分钟):**

```
"技术栈总结:

硬件:
- ESP32 (WiFi + GPIO)
- I2C传感器 (BME680, OLED)
- UART传感器 (PMS5003)

软件:
- MicroPython (ESP32端)
- Python (服务器端)
- MQTT (通信协议)
- InfluxDB (时序数据库)
- Grafana (可视化)

关键技术点:
- MQTT发布/订阅模型
- 时序数据库设计
- 实时数据可视化"
```

**项目亮点(1分钟):**

```
"本项目的主要亮点:

1. 完整的IoT数据流
   从采集→传输→存储→可视化

2. 工业级通信协议
   使用MQTT标准协议

3. 可扩展架构
   易于添加新传感器

4. 低成本实现
   总成本 < ¥300"
```
