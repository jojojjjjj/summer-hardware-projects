## 🎯 详细评分细则 | Detailed Scoring Criteria

### 一、技术实现 (40分)

#### 1.1 功能完整性 (15分)

| 分数 | 标准 | 评估要点 |
|------|------|---------|
| **13-15分** | 优秀 | 所有计划功能实现,运行稳定,无重大bug |
| **10-12分** | 良好 | 核心功能实现,有少量问题但不影响使用 |
| **7-9分** | 及格 | 基本功能实现,存在明显问题需要修复 |
| **0-6分** | 不及格 | 功能不完整或无法运行 |

**功能检查清单:**
- [ ] ESP32能正常启动和运行
- [ ] BME680能读取温湿度/气压数据
- [ ] PMS5003能读取PM2.5/PM10数据
- [ ] OLED能显示传感器数据
- [ ] WiFi能稳定连接
- [ ] MQTT能正常上传数据
- [ ] 服务器能接收并存储数据
- [ ] Grafana能可视化显示数据

#### 1.2 代码质量 (15分)

| 分数 | 标准 | 评估要点 |
|------|------|---------|
| **13-15分** | 优秀 | 代码规范、结构清晰、有完整注释、模块化好 |
| **10-12分** | 良好 | 代码较规范、有基本注释、结构合理 |
| **7-9分** | 及格 | 代码可用但不够规范、注释不足 |
| **0-6分** | 不及格 | 代码混乱、难以维护、无注释 |

**代码质量检查:**
```python
# 优秀代码示例
class SensorManager:
    """传感器管理器 - 整合所有传感器读取"""
    
    def __init__(self, i2c, uart):
        """初始化传感器管理器"""
        self.i2c = i2c
        self.uart = uart
        self.bme = None
        self._init_bme()
    
    def read_temperature(self):
        """读取温度数据
        
        Returns:
            float: 温度值(摄氏度),读取失败返回None
        """
        try:
            self.bme.get_sensor_data()
            return self.bme.data.temperature
        except Exception as e:
            print(f"温度读取错误: {e}")
            return None
```

**评分要点:**
- 命名规范(变量/函数/类名)
- 代码缩进和格式
- 注释完整度
- 模块化程度
- 错误处理

#### 1.3 系统稳定性 (10分)

| 分数 | 标准 | 评估要点 |
|------|------|---------|
| **9-10分** | 优秀 | 长时间运行无崩溃,异常处理完善,自动恢复 |
| **7-8分** | 良好 | 偶有小问题,能自动恢复,不影响使用 |
| **5-6分** | 及格 | 有明显bug,需要手动重启,基本可用 |
| **0-4分** | 不及格 | 频繁崩溃或无法稳定运行 |

**稳定性测试:**
- 连续运行1小时无崩溃
- 网络断开后能自动重连
- 传感器异常能正常处理
- 内存无泄漏
