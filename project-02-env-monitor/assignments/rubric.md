### 一、技术实现 (40分)

#### 1.1 功能完整性 (15分)

**评分细则:**

| 功能 | 分数 | 验收标准 |
|------|------|---------|
| **ESP32基础功能** | 2分 | 能正常启动、运行、响应命令 |
| **BME680读取** | 2分 | 能读取温湿度/气压,数据准确 |
| **PMS5003读取** | 2分 | 能读取PM2.5/PM10,数据准确 |
| **OLED显示** | 2分 | 能显示传感器数据,界面清晰 |
| **WiFi连接** | 2分 | 能稳定连接WiFi,自动重连 |
| **MQTT上传** | 3分 | 能上传数据,格式正确,稳定可靠 |
| **服务器接收** | 1分 | 订阅器能接收并解析数据 |
| **数据存储** | 1分 | InfluxDB能正常写入数据 |

**扣分项:**
- 功能缺失: -2分/项
- 功能不稳定: -1分/项
- 数据错误: -1分/项

#### 1.2 代码质量 (15分)

**评分细则:**

| 方面 | 分数 | 评分标准 |
|------|------|---------|
| **结构设计** | 5分 | 模块化清晰,职责分明,易于维护 |
| **命名规范** | 3分 | 变量/函数/类名符合规范 |
| **注释完整** | 4分 | 所有函数有docstring,关键逻辑有注释 |
| **错误处理** | 3分 | 有完善的异常处理和错误日志 |

**代码质量检查:**

```python
# 优秀代码示例 (5分)
class SensorManager:
    """传感器管理器 - 整合所有传感器读取

    Attributes:
        bme: BME680传感器实例
        pm: PMS5003传感器实例

    Methods:
        read_all: 读取所有传感器数据
        get_status: 获取传感器状态
    """

    def __init__(self, i2c, uart):
        """初始化传感器管理器

        Args:
            i2c: I2C总线实例
            uart: UART实例
        """
        self.i2c = i2c
        self.uart = uart
        self._init_sensors()

    def read_all(self):
        """读取所有传感器数据

        Returns:
            SensorReading: 包含所有传感器数据的对象
        """
        reading = SensorReading()
        try:
            # 读取BME680
            temp, hum, pres = self.bme.read()
            reading.temperature = temp
            reading.humidity = hum
            reading.pressure = pres

            # 读取PMS5003
            pm25, pm10 = self.pm.read()
            reading.pm25 = pm25
            reading.pm10 = pm10

            reading.valid = self._validate(reading)
            return reading

        except Exception as e:
            logger.error(f"传感器读取错误: {e}")
            reading.valid = False
            return reading
```
