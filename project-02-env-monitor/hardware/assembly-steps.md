## 🔧 组装流程 | Assembly Process

### 阶段1: 面包板原型 | Breadboard Prototype (Day 1-6)

**目标:** 快速验证电路设计

**步骤:**
1. 将ESP32插入面包板中央
2. 按接线图连接所有传感器
3. 使用杜邦线连接
4. 测试所有功能
5. 确认无误后继续

**预期结果:**
- 所有传感器正常读取
- OLED显示正常
- 系统稳定运行

---

### 阶段2: 焊接原型板 | Solder Prototype Board (Day 10)

**目标:** 创建永久电路

**步骤:**

1. **规划布局**
   ```
   在万能板上规划:
   - ESP32位置(中央)
   - 传感器接口位置(边缘)
   - 电源接口位置(一角)
   ```

2. **安装排母**
   ```
   将排母焊接到万能板:
   - ESP32插座(18×2)
   - 传感器接口(4×1)
   - 电源接口(2×1)
   ```

3. **焊接连接**
   ```
   按接线图焊接:
   - 电源线(较粗)
   - 信号线(较细)
   - GND连接(公共地)
   ```

4. **测试连通性**
   ```
   用万用表测试:
   - VCC到GND无短路
   - 每条连接导通
   - 引脚连接正确
   ```

5. **安装元件**
   ```
   将ESP32插入插座
   连接传感器
   上电测试
   ```

**⚠️ 安全提示:**
- 焊接时保持通风
- 电烙铁远离易燃物
- 使用烙铁架
- 烙铁温度300-350°C

---

### 阶段3: 外壳组装 | Enclosure Assembly (Day 10-11)

**目标:** 完成产品外观

**步骤:**

1. **安装电路板**
   ```
   - 将电路板放入外壳
   - 用螺丝固定
   - 确认接口对齐
   ```

2. **安装传感器**
   ```
   - BME680固定在侧面
   - PMS5003固定在底部
   - OLED固定在前面板
   ```

3. **整理线缆**
   ```
   - 用扎带固定线缆
   - 留出适当长度
   - 避免遮挡散热
   ```

4. **安装外壳**
   ```
   - 安装侧板
   - 安装前面板
   - 安装顶盖
   - 检查所有对齐
   ```

5. **最终测试**
   ```
   - 上电测试
   - 检查显示
   - 验证功能
   ```

## 🎯 质量检查 | Quality Check

### 检查清单

**电气检查:**
- [ ] 无短路(VCC-GND)
- [ ] 电源电压正确(3.3V/5V)
- [ ] 所有引脚导通
- [ ] 无冷焊

**功能检查:**
- [ ] ESP32启动正常
- [ ] OLED显示正常
- [ ] BME680读数正常
- [ ] PMS5003读数正常
- [ ] WiFi连接正常

**外观检查:**
- [ ] 无明显缝隙
- [ ] 无松动部件
- [ ] 标识清晰
- [ ] 整体美观

## 🐛 故障排查 | Troubleshooting

| 问题 | 可能原因 | 解决方案 |
|------|---------|---------|
| 设备不工作 | 电源接反 | 检查VCC/GND |
| ESP32发热 | 有短路 | 断电检查 |
| OLED无显示 | I2C接线错误 | 检查SDA/SCL |
| PM读数异常 | 波特率不对 | 设置为9600 |
| WiFi连不上 | 天线接触不良 | 检查天线 |
