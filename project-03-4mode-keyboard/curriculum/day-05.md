### USB 协议基础 | USB Protocol Fundamentals

USB（Universal Serial Bus）是FantasyKB有线连接电脑的核心接口。

#### USB 设备类型（设备类 Device Class）

| 设备类 | 名称 | FantasyKB 中的应用 |
|---|---|---|
| **HID** | Human Interface Device | 键盘、鼠标——即插即用，无需驱动 |
| **CDC** | Communication Device Class | 串口（调试用） |
| **Mass Storage** | 大容量存储 | U盘模拟（GL823K芯片） |

#### FantasyKB USB 架构

```
电脑 USB 口
     |
  USB Type-C 母座
     |
  +-----+------+
  | SL2.1A Hub  |  <-- USB Hub芯片，将1个USB口扩展为多个
  +-----+------+
    |    |    |
    |    |    +--> GL823K --> Flash (模拟U盘，存储配置文件)
    |    |
    |    +--> 模拟开关 U6/U7 --> ESP32-S3 USB (键盘主控)
    |
    +--> TYPE-A 口 (可插鼠标/USB设备)
           |
           通过模拟开关切换：
           - 无线模式时: TYPE-A 鼠标数据 -> ESP32-S3 -> 无线转发
           - USB模式时:  TYPE-A 直连 Hub -> 电脑
```

**模拟开关的作用：**
FantasyKB 使用模拟开关（U6/U7）在两种模式间切换 USB 连接：
- **无线模式**（BLE/WiFi/ESP-NOW）：鼠标连接到ESP32-S3，数据通过无线转发给电脑
- **USB有线模式**：鼠标和ESP32-S3都挂在Hub上，直连电脑

### 电源管理架构 | Power Architecture

FantasyKB 使用3.7V锂电池供电，需要多个电源管理芯片协同工作：

```
                     FantasyKB 电源架构

USB 5V ──────+──> TP4056 ──> 锂电池充电 (3.7V)
              |              (恒流/恒压充电)
              |
              +──> (USB直供5V)

锂电池 3.7V ─┬─> HX3608 (DC-DC升压) ──> 5V
             |                           |
             |                    +------+------+------+
             |                    |      |      |      |
             |                 WS2812B  Hub  功放  USB口
             |
             +──> ME6217 (LDO线性稳压) ──> 3.3V
                                              |
                                     +----+----+----+----+
                                     |    |    |    |    |
                                   ESP32 音频 编码器 显示屏 其他
```

#### 各芯片功能详解

| 芯片 | 类型 | 功能 | 关键参数 |
|---|---|---|---|
| **TP4056** | 锂电池充电IC | 将USB 5V转为恒流/恒压给电池充电 | 充电电流可设（通常500mA-1A） |
| **HX3608** | DC-DC升压 | 将电池3.7V升压到5V | 效率约85-92% |
| **ME6217** | LDO线性稳压 | 将5V或电池电压降压到3.3V | 压差仅200mV，最大500mA |

**为什么要多个电源芯片？**
- ESP32-S3、编解码器、传感器需要 3.3V 稳定电源 -> **ME6217 LDO**
- WS2812B LED、USB设备需要 5V -> **HX3608 升压**
- 电池需要安全充电 -> **TP4056 充电IC**

### 锂电池安全 | Battery Safety

**锂电池安全守则（必须遵守！）：**

| 规则 | 原因 |
|---|---|
| 不要短路电池正负极 | 短路电流可达数十安培，会起火 |
| 不要过充（超过4.2V） | 电池膨胀、起火 |
| 不要过放（低于2.7V） | 电池永久损坏 |
| 不要穿刺、挤压 | 内部短路，起火 |
| 充电时有人看管 | 异常及时处理 |
| 使用保护板（BMS） | TP4056内置保护，但仍需注意 |

CH342F 是一颗双通道 USB-TTL 桥接芯片，FantasyKB 用它将两个串口转成USB：
- 通道1：连接键盘主控 ESP32-S3（固件下载 + 调试）

**测试步骤：**

4. 设置波特率 115200，发送测试数据
5. 观察模块的 TX/RX 指示灯是否闪烁

### 任务 5.4: TP4056 充电模块实验

**安全提醒：锂电池操作必须遵守安全守则！**

**实验步骤：**
1. **认识 TP4056 模块**：找到充电指示灯（红=充电中，蓝=充满）
2. **连接电池**：将 3.7V 锂电池接到 TP4056 的 BAT+ 和 BAT- 端
3. **开始充电**：通过 USB 口给 TP4056 供电
4. **测量充电电压**：用万用表测量电池两端电压
   - 充满时电压应为 4.2V
   - 放电截止电压约 2.7-3.0V
5. **计算充电时间**：电池容量 / 充电电流 = 充电时间

```
例: 2000mAh 电池, TP4056 充电电流 500mA
充电时间 ≈ 2000/500 x 1.2 ≈ 4.8 小时
(乘1.2是因为恒压阶段充电变慢)
```

## 常见问题 | FAQ

**Q: USB Hub 接上后电脑不识别怎么办？**
A: (1) 检查USB线是否支持数据传输 (2) 确认SL2.1A供电是否正常 (3) 检查上行口（接电脑）和下行口（接设备）是否正确连接。

**Q: TP4056 充电指示灯一直红灯不转蓝？**
A: 可能原因：(1) 电池电压过低（需要预充电阶段）(2) 电池老化 (3) 充电电流设置过大导致无法达到恒压阶段。耐心等待几小时。

**Q: HX3608 升压输出不到 5V？**
A: 检查输入电压是否过低（低于2.7V可能无法正常升压），检查输出电容是否足够，确认电感规格是否符合要求。
