## 上午：SMD 焊接技术 | Morning: SMD Soldering Techniques

### 为什么要学这个? | Why Learn This?

本项目的 12 块 PCB 上有大量贴片元件（SMD）——电阻电容（0805 封装）、IC 芯片（QFN 封装）、LED（WS2812B 2020 封装）。与 Day 10 的通孔焊接不同，贴片焊接需要更精细的操作和不同的工具（锡膏、钢网、热风枪或回流焊）。掌握 SMD 焊接是从"爱好者"进阶到"工程师"的关键技能。

This project's 12 PCBs have many SMD components — 0805 passives, QFN ICs, WS2812B LEDs. Unlike through-hole soldering (Day 10), SMD requires finer techniques and different tools. Mastering SMD is key to advancing from "hobbyist" to "engineer."

### 任务 19.1: SMD 焊接方法对比 (30 分钟)

**两种 SMD 焊接方法 Two SMD Soldering Methods:**

```
方法一: 锡膏 + 回流焊 (推荐，适合批量)
  1. 用钢网在 PCB 焊盘上刮涂锡膏
  2. 用镊子放置元件
  3. 放入回流焊炉（或热风枪加热）
  4. 锡膏熔化，表面张力自动对位

  ┌──────┐    ┌──────────┐    ┌──────────┐
  │ 钢网  │──→│ 刮涂锡膏  │──→│ 放置元件   │──→ 回流焊
  └──────┘    └──────────┘    └──────────┘

方法二: 烙铁 + 焊锡丝 (适合少量/修补)
  1. 先在一个焊盘上锡（少量）
  2. 用镊子夹住元件，烙铁融化锡固定一端
  3. 调整对位，焊接另一端
  4. 检查是否虚焊

  适合: 0805 电阻电容，SOIC 封装 IC
  困难: QFN 封装（引脚在底部，无法直接接触）
```

**锡膏印刷回流焊工艺 Reflow Soldering Process:**

```
温度曲线 Temperature Profile:

  温度 ↑
       │         ┌──────┐  峰值 235-250°C (10-30s)
       │        /        \
       │       /          \
       │  ────/            \────
       │  预热   浸润    回流   冷却
       │ 150°C  180°C  235°C
       └──────────────────────────→ 时间
         60-90s  60-90s  30-60s  冷却

  锡膏成分: Sn96.5/Ag3.0/Cu0.5 (SAC305, 无铅)
  熔点: ~217°C
```

### 任务 19.2: QFN 芯片焊接演示 (45 分钟)

**QFN 封装焊接要点 QFN Soldering Notes:**

```
QFN (Quad Flat No-leads) 封装特点:
  - 引脚在芯片底部四周
  - 底部有裸露的散热焊盘 (Thermal Pad)
  - 无法从侧面看到引脚
  - 需要锡膏 + 回流焊

ESP32-S3-WROOM-1 模块:
  - 已集成在模组上，焊接模组的排针/焊盘即可
  - 不需要直接焊接裸芯片 (太好了！)

TP4056 / ME6217 / SL2.1A:
  - SOT-23-5 或 MSOP 封装
  - 引脚间距 0.95mm，烙铁可以焊接

74HC165:
  - SOIC-16 封装
  - 引脚间距 1.27mm，烙铁可以焊接

ES7210 / ES8311:
  - QFN 封装 (引脚间距 0.4mm)
  - 必须使用锡膏 + 回流焊！
  - 需要放大镜或显微镜检查
```

**烙铁焊接 QFN 替代方案（应急用）:**
1. 在焊盘上涂一层薄薄的助焊剂
2. 用烙铁在每个焊盘上预先镀一层薄锡
3. 放置芯片，用热风枪从上方均匀加热
4. 表面张力会自动对齐芯片

### 任务 19.5: 安装热插拔轴座和轴体 (60 分钟)

**安装步骤 Installation Steps:**

```
1. 焊接热插拔轴座 (如果还没焊):
   - 轴座平贴 PCB
   - 先焊一个脚，检查是否平整
   - 再焊另一个脚
   - 用轴体试插，确认松紧合适

2. 安装轴体:
   - 将机械轴体对准轴座
   - 垂直用力按下
   - 听到"咔"声表示安装到位
   - 拔出检查是否牢固

3. 安装键帽:
   - 将键帽对准轴体十字柱
   - 轻轻按下
   - 检查高度是否一致

4. 安装旋钮:
   - 将 FOC 电机组件安装到旋钮板
   - 连接磁编码器排线
   - 安装圆形 LCD 屏幕
   - 连接旋钮板到主控板（磁吸接口或排线）
```

## 日终总结 | Daily Wrap-up (16:30-17:00)

**今日关键词 Key Takeaways:**
1. SMD 焊接有两种主流方法：锡膏回流焊（批量）和烙铁手焊（修补）
2. QFN 封装必须回流焊，SOIC/SOT 可以烙铁焊
3. 焊接顺序：先贴片 IC → 再贴片被动元件 → 最后通孔/大元件
4. 通电前必须做短路检查——这是防止烧板的关键步骤
