### 第三部分：技术讲解 (5 分钟)

**目标**：展示你对核心技术的理解深度

**必讲内容 Required Topics:**

1. **硬件架构**（2 分钟）
   - ESP32-S3 主控 + 74HC165 菊花链（80 键扫描）
   - 电源树设计（TP4056 + HX3608 + ME6217）
   - 12 块 PCB 设计（4 层板、阻抗控制）
   - USB 子系统（SL2.1A Hub + CH342F + GL823K）

2. **固件架构**（2 分钟）
   - ESP-IDF + FreeRTOS 多任务架构
   - 键盘扫描 + 消抖算法
   - 4 种无线模式（USB/BLE/WiFi/ESP-NOW）
   - 音频管道（ES7210 + ES8311 + I2S）

3. **创新点**（1 分钟）
   - FOC 力反馈旋钮（TMC6300 + AS5047P + SimpleFOC）
   - 磁吸扩展系统
   - LVGL 旋钮界面

**PPT 建议**：5-6 页，每页有图示或示意图

### 第四部分：实物演示 (5 分钟)

**目标**：让观众亲眼看到你的作品

**演示流程 Demo Script:**

```
1. USB 有线模式 (30s)
   打开记事本，逐键输入
   "USB 有线模式延迟最低，适合日常打字"

2. BLE 蓝牙模式 (30s)
   切换到 BLE 模式（Fn+2），配对手机/平板
   "蓝牙模式适合连接移动设备"

3. WiFi 键盘模式 (30s)
   切换到 WiFi 模式（Fn+3），展示远程打字
   "WiFi 模式适合远程控制"

4. ESP-NOW 2.4GHz 模式 (30s)
   切换到 ESP-NOW（Fn+4），插入接收器
   "2.4GHz 模式低延迟，类似无线鼠标"

5. 旋钮演示 (30s)
   旋转调节音量，按下静音
   展示 LCD 显示屏
   "力反馈旋钮支持 Surface Dial 协议"

6. RGB 灯效 (30s)
   展示不同模式对应不同颜色
   "蓝色=USB，绿色=BLE，橙色=WiFi，紫色=ESP-NOW"

7. 语音交互 (30s)
   对麦克风说话，展示音频采集
   "支持语音交互，通过音频子系统采集和播放"

8. 磁吸扩展 (30s)
   连接/断开子板
   "磁吸接口支持热插拔扩展"
```

**演示备份方案 Backup Plan:**
- 如果 BLE 配对失败，切换到 USB 有线模式继续
- 如果 ESP-NOW 信号弱，移近接收器
- 准备功能演示视频作为最终备份

## PPT 制作建议 | PPT Design Tips

| 原则 Principle | 说明 Explanation |
|---|---|
| 少字多图 | 每页不超过 5 行文字，多用图示和照片 |
| 代码精选 | 只展示最关键的代码片段（不超过 10 行） |
| 配色统一 | 使用一致的配色方案 |
| 字体大小 | 标题 >= 28 号，正文 >= 20 号 |
| 动画适度 | 适当使用动画，但不要花哨 |
