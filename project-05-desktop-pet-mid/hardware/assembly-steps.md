# 组装步骤 | Assembly Steps

## 概述 | Overview

本文档详细指导 ESP-SparkBot AI 桌面机器人的完整组装流程。整个过程包含 PCB 焊接、飞线连接、3D 打印外壳安装和固件烧录，预计耗时 2-3 天（含等待 PCB 和 3D 打印的时间）。

This document provides a detailed, step-by-step assembly guide for the ESP-SparkBot AI desktop robot. The process covers PCB soldering, wire routing, 3D printed enclosure assembly, and firmware flashing. Estimated hands-on time: 2-3 days (excluding PCB fabrication and 3D printing lead time).

---

## 工具准备 | Tools Preparation

在开始之前，请确保以下工具齐备：

| 工具 | 用途 | 备注 |
|------|------|------|
| 水口钳 | 剪开拼板、修剪元件引脚 | 必备 |
| PCB 夹 / 焊接夹具 | 固定 PCB 方便焊接 | 推荐 |
| 锡浆 (针筒装) + 锡丝 | 贴片焊接 + 插件焊接 | 锡浆推荐低温无铅 |
| 热风枪 + 电烙铁 | 贴片回流焊 + 手工补焊 | 烙铁推荐 T12/K 型头 |
| 镊子 (直头+弯头) + 元件盒 | 夹取和分类贴片元件 | 小元件极易丢失 |
| 飞线 (软线，28AWG) | 底盘 PCB 之间的飞线连接 | 不要用硬线 |
| 万用表 | 测电压、测短路、蜂鸣档导通测试 | **每一步焊接后必测** |
| 助焊膏 / 助焊剂 | 辅助 QFN 和密脚 IC 焊接 | 焊后需用洗板水清洗 |
| 洗板水 (无水乙醇可替代) | 清洗焊后残留助焊剂 |  |
| B-700 胶水 | 固定易脱落的接插件和排线 | 点少量即可 |
| KN95 口罩 + 护目镜 | 焊接烟雾防护 + 防锡珠飞溅 | 安全第一 |
| USB-C 数据线 (支持数据传输) | 烧录固件和串口调试 | 不要用充电线 |

## 步骤 1：PCB 打板与检查 | Step 1: PCB Fabrication & Inspection

### 1.1 嘉立创下单 | JLCPCB Ordering

1. 登录嘉立创 (https://www.jlcpcb.com)，上传 Gerber 文件。
2. **板厚选择 1.0mm**（默认 1.6mm，必须手动改）。
3. **每个板子单独下单**（核心板、底板、底盘 PCB 套件各一单），因为每个板的层数和要求不同。
4. 颜色推荐：黑色（搭配外壳更协调）。
5. 预计交期：2-4 天（标准交期）。

### 1.2 到货检查 | Inspection After Receiving

收到 PCB 后，**不要急于焊接**，先做以下检查：

```
检查清单：
□ 1. 核对 PCB 数量：核心板 ×1、底板 ×1、底盘 PCB ×4
□ 2. 目视检查：无划痕、无断线、无露铜
□ 3. 万用表蜂鸣档：测试 Type-C 的 VCC 和 GND 是否短路
    - 如果蜂鸣（短路），说明 PCB 有缺陷，联系嘉立创补发
□ 4. 用万用表测量电源层和地层之间是否短路（正常应为开路）
□ 5. 检查板厚是否为 1.0mm（与 1.27mm 排针排母试配）
```

### 1.3 剪板与打磨 | Depanel & Deburring

- 嘉立创的板子通常是拼板（V-Cut 连接），用水口钳沿 V-Cut 槽剪开。
- 剪开后断面粗糙，用细砂纸（400-600目）轻轻打磨边缘，避免毛刺划伤外壳。
- 打磨后用水冲洗，晾干或吹干。

## 步骤 2：焊接 — 核心板 | Step 2: Soldering — Main Board

> **核心原则：每焊完一组电路，立即用万用表测试，确认无误再焊下一组。先易后难，先电源后信号。**

### 2.1 先焊 DC-DC 电源电路 | Solder Power Supply First

1. 焊接 DC-DC 降压芯片 (5V → 3.3V) 及其外围电阻电容。
2. 焊接完成 → **万用表测 3.3V 输出**：
   - 通过 Type-C 供电（5V），测量 3.3V 测试点。
   - 正常读数为 3.3V ± 0.1V。
   - **如果输出 0V 或 5V**：检查 DC-DC 焊接方向、外围电阻值是否正确。
3. 如果电压正常，断开电源，继续下一步。

### 2.2 焊主控模组 | Solder ESP32-S3 Module

1. 将 ESP32-S3-WROOM-1-N16R8 模组对齐 PCB 焊盘。
2. 先在模组对角两个引脚上锡固定位置。
3. 确认模组完全平整、无歪斜后，依次焊接剩余引脚。
4. 用万用表蜂鸣档测试每个引脚与相邻引脚是否短路（密脚容易连锡）。
5. **通电验证**：插入 USB-C，ESP32-S3 的电源指示灯应亮起。

### 2.3 焊摄像头 FPC 座 | Solder Camera FPC Connector

1. 焊接 OV2640 的 FPC 插座（DVP 接口）。
2. 注意方向：金属触点面朝向与 PCB 丝印一致。
3. 焊完后用放大镜检查密脚是否有连锡。

### 2.4 焊显示屏排线座 | Solder Display Connector

1. 焊接 1.54寸 ST7789 屏幕的 SPI 排线座。
2. **屏幕排线非常脆弱**，焊接和后续插拔时要格外小心。
3. 排线座焊好后，先不要插屏幕，等全部焊完再插。

### 2.5 焊音频电路 | Solder Audio Circuit

焊接顺序：
1. ES8311 音频编解码 IC（QFN 封装，用锡浆 + 热风枪）
2. NS4150B 功放 IC
3. 外围电阻电容
4. 3020 方形腔体喇叭（注意正负极）
5. B4013AM422-42 驻极体麦克风（注意正负极）

焊后验证：暂不测试，等固件烧录后一起测试。

### 2.6 焊排针 | Solder Header Pins

1. 焊接 1.27mm 间距排针（用于与底板堆叠连接）。
2. 确保排针**垂直**，不歪斜，否则插入底板时会卡住。
3. 排针焊好后，试插底板确认位置对齐。

---

## 步骤 3：焊接 — 底板 | Step 3: Soldering — Bottom Board

### 3.1 焊充电电路 | Solder Charging Circuit

1. 焊接充电管理 IC + 外围元件。
2. 焊接 USB-C 接口：
   - USB-C 16P 母座引脚密集，建议先用锡浆涂焊盘，热风枪回流。
   - **焊完必测**：万用表蜂鸣档测试 VCC-GND 是否短路。
3. 通电测试：插 USB-C，测量充电 IC 输出端是否有电压。

### 3.2 焊 IMU | Solder BMI270

1. BMI270 为 LGA 封装，使用锡浆 + 热风枪焊接。
2. 注意芯片方向：PCB 丝印上通常有 Pin 1 标记（小圆点）。
3. 焊完后用放大镜检查底部焊盘是否有桥接。

### 3.3 焊磁吸连接器 | Solder Magnetic Pogo-Pin Connector

1. 焊接底部 4P pogopin 母座。
2. 确保座子与 PCB 平面平行，不翘起。
3. 用 B-700 胶水在座子边缘点胶加固（磁吸插拔有力度）。

### 3.4 焊触摸 FPC | Solder Touch FPC

1. 焊接 FPC 软排线触摸按键（顶部 + 两侧触摸区域）。
2. FPC 排线较软，焊接时温度不要太高（300°C 以下）。
3. 确认排线方向正确。

### 3.5 焊排母 | Solder Receptacle

1. 焊接 1.27mm 间距排母（对应核心板的排针）。
2. 确保排母**垂直**，与核心板排针对齐。

## 步骤 4：焊接 — 履带底盘 | Step 4: Soldering — Tracked Chassis

履带底盘包含 4 块 PCB：主控板、前灯板、后灯板、磁吸板。它们需要通过飞线连接。

### 4.1 焊各 PCB 上的元件 | Solder Components on Each PCB

1. **主控板**：焊 ESP8684-WROOM-02C-N2 模组 + 电机驱动电路 + 外围元件。
2. **前灯板**：焊 LED 灯珠。
3. **后灯板**：焊 LED 灯珠。
4. **磁吸板**：焊 4P pogopin 公头（对应底板母座）。

### 4.2 飞线连接 4 块 PCB | Wire Routing Between PCBs

- **参考开源原理图的飞线图**，千万不要凭感觉接。
- 使用 **28AWG 软飞线**（硅胶线最好），不要用硬线（容易折断）。
- 每根飞线两端留 5-10mm 余量，不要太紧绷。
- 飞线焊完后用热熔胶或 B-700 胶水在线束路径上点胶固定，防止震动脱落。
- **通电前必测**：万用表蜂鸣档测试各 PCB 之间的 VCC-GND 不短路。

### 4.3 安装电机和履带 | Install Motors and Tracks

1. 将 N20 减速电机装入底盘电机座。
2. 安装驱动轮到电机轴上（可能需要滴少量螺丝胶防松）。
3. 套上橡胶履带，检查松紧度（用手指轻按履带中部，下陷 2-3mm 为合适）。

---

## 步骤 5：3D 打印外壳 | Step 5: 3D Printed Enclosure

### 5.1 获取 STL 文件 | Get STL Files

STL 文件已随项目开源，包含以下打印件：

| 打印件 | 数量 | 说明 |
|--------|------|------|
| 主体后壳 | 1 | 容纳核心板 + 底板 |
| 前盖（带屏幕卡口） | 1 | 屏幕窗口 + 摄像头孔 |
| 摄像头前盖 | 1 | 保护摄像头镜头 |
| 摄像头后盖 | 1 | 固定摄像头模组 |
| 履带驱动轮 ×2 + 从动轮 ×2 | 4 | 底盘轮子 |

### 5.2 打印选项 | Printing Options

| 方式 | 材料 | 价格 | 效果 |
|------|------|------|------|
| 嘉立创 3D 打印 | 树脂 9600 | 30-50元 | 表面光滑、细节好、精度高 |
| 自行 FDM 打印 | PLA | 10-20元 | 成本低、表面有层纹、需后处理 |
| 淘宝代打 | 树脂 | 40-60元 | 适合没打印机的同学 |

推荐使用嘉立创树脂 9600 材料打印，表面质感好，摄像头孔位精度高。

### 5.3 打印件检查 | Check Printed Parts

- 检查屏幕卡口尺寸：1.54寸屏幕应能恰好嵌入前盖卡口。
- 检查摄像头孔位：OV2640 镜头能否从孔中伸出。
- 检查主体后壳与底板的安装孔位是否对齐。

## 步骤 6：总装 | Step 6: Final Assembly

### 6.1 核心板 + 底板堆叠 | Stack Main Board + Bottom Board

1. 将核心板的 1.27mm 排针插入底板的 1.27mm 排母。
2. **务必对照 PCB 丝印方向标记**，确认方向正确后再用力按下。
3. 万用表蜂鸣档测试 VCC-GND：如果蜂鸣，说明方向插反或焊接有短路，**立即断开**。
4. 堆叠后的两块板应该紧密贴合、无歪斜。

### 6.2 装入外壳 | Install into Enclosure

1. 将堆叠好的 PCB 装入主体后壳，对准螺丝孔。
2. 安装前盖：
   - 先将屏幕和摄像头从内部对准前盖孔位。
   - 注意**摄像头方向**：OV2640 排线应从特定方向弯折，参考原理图中的安装示意图。
   - 盖上摄像头前盖和后盖，M2 螺丝固定。
3. 用 M2 螺丝将前盖与后壳固定。

### 6.3 安装电池 | Install Battery

1. 将 3.7V 2000mAh 聚合物锂电池放入主体后壳的电池仓。
2. 插头插入底板的电池插座（1.25mm 红黑插头）。
   - **注意正负极**：红正黑负，反接会烧充电 IC。
3. 固定电池（可用双面胶粘在壳内）。

### 6.4 安装履带底盘 (可选) | Install Tracked Chassis (Optional)

1. 将履带底盘的磁吸板对准底板底部的 4P pogopin 母座，磁力自动吸附。
2. 确认吸附牢固：轻轻晃动底盘，不应脱落。
3. 磁吸接口同时提供供电和 UART 通信，无需额外接线。

## 步骤 7：固件烧录与测试 | Step 7: Firmware Flashing & Testing

### 7.1 烧录 factory_demo 固件 | Flash Factory Demo Firmware

1. 用 USB-C 数据线连接 ESP-SparkBot 和电脑。
2. **首次烧录**：
   - **按住 BOOT 键不放**，再插入 USB → 电脑识别到串口后松手。
   - 如果直接插 USB 不识别 COM 口，就是没进下载模式。
3. 使用 ESP-IDF 或 PlatformIO 编译并烧录 `factory_demo` 固件：
   ```bash
   # ESP-IDF 方式
   idf.py -p COMx flash monitor
   
   # PlatformIO 方式
   pio run -t upload -t monitor
   ```
4. 烧录成功后，串口监视器应输出启动日志。

### 7.2 功能测试清单 | Functional Test Checklist

| 测试项 | 预期结果 | 问题定位 |
|--------|----------|----------|
| 屏幕 | 显示 UI 界面 (240×240) | 黑屏 → 检查 SPI 焊接；花屏 → 检查 spi_mode |
| 摄像头 | 显示实时画面 | 无画面 → 检查 FPC 排线方向 |
| 音频播放 | 发出声音 (TTS 播报) | 无声 → 检查 NS4150B 和喇叭 |
| 麦克风 | 语音识别正常工作 | 不收音 → 检查 ES8311 和咪头 |
| 触摸按键 | 触发对应功能 | 不灵敏 → 需校准基线 |
| WiFi | 连接 2.4GHz WiFi | 注意 ESP32-S3 不支持 5GHz |
| BMI270 | 姿态数据正确 | I2C 地址扫描确认 |
| 电池充电 | Type-C 供电时充电指示灯亮 | 不亮 → 检查充电 IC |

### 7.3 测试 AI 对话功能 | Test AI Conversation

1. 连接 WiFi 后，按下指定触摸区域唤醒语音对话。
2. 确认网络通畅（需要外网访问 LLM API）。
3. 测试 TTS 语音合成和 STT 语音识别。

```
□ 万用表测试所有电源节点电压正常 (3.3V, 5V)
□ VCC-GND 无短路
□ 屏幕正常显示 UI
□ 摄像头画面清晰
□ 音频输入输出正常
□ 触摸按键响应正常
□ WiFi 连接正常
□ 电池可充可放
□ 底盘运动正常 (如有)
□ 所有螺丝拧紧，外壳无晃动
□ 固件已烧录最新版本
```
