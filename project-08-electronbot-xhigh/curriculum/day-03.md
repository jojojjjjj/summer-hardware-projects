## 上午: 焊接基础与舵机驱动板 | Morning: Soldering Basics & Servo Boards

### 为什么要学这个? | Why Learn This?

ElectronBot的PCB上有多种封装的贴片芯片——从相对简单的TSSOP-20到极具挑战性的QFN-32。QFN（Quad Flat No-leads）封装没有外露的引脚，焊盘在芯片底部，对焊接技巧要求很高。掌握QFN焊接是成为硬件高手的标志性技能。

ElectronBot's PCBs feature multiple SMD package types -- from the relatively simple TSSOP-20 to the highly challenging QFN-32. QFN (Quad Flat No-leads) packages have no exposed leads; the pads are under the chip, demanding advanced soldering skills. Mastering QFN soldering is a hallmark skill for hardware proficiency.

今天焊接的每一块板子都将在后续课程中使用——舵机驱动板控制机器人的每个关节，主控板是整个系统的大脑。焊接质量直接决定了你的ElectronBot能否正常工作。

Every board you solder today will be used in subsequent sessions -- servo driver boards control each joint, and the main board is the system's brain. Soldering quality directly determines whether your ElectronBot will work properly.

### 任务3.1: 焊接工具与安全 (20分钟)

**安全第一！焊接涉及高温（250-400度C）和有毒烟雾，必须严格遵守安全规则。**

Safety first! Soldering involves high temperatures (250-400C) and toxic fumes. Strictly follow these safety rules.

```
焊接安全守则:

1. 电烙铁永远放在烙铁架上 -- 烙铁头温度超过300度C
   Always keep the iron in its stand -- tip temperature exceeds 300C

2. 手只握烙铁手柄（黑色塑料部分），绝不触碰金属部分
   Only hold the handle (black plastic part), never touch the metal

3. 焊接时戴防护眼镜（焊锡可能飞溅）
   Wear safety glasses (solder may splash)

4. 焊接区域保持通风 -- 开启排风扇或打开窗户
   Keep the soldering area ventilated -- turn on exhaust fan

5. 助焊膏含化学物质，避免皮肤直接接触，使用后洗手
   Flux contains chemicals; avoid skin contact, wash hands after use

6. 不要穿短裤和凉鞋（焊锡掉落会烫伤）
   Don't wear shorts or sandals (falling solder causes burns)

7. 热风枪温度可达400度C，使用时远离他人和易燃物
   Hot air gun can reach 400C; keep away from others and flammable items

8. 离开座位时关掉电烙铁和热风枪电源
   Turn off iron and hot air gun when leaving
```

### 任务3.2: QFN封装焊接技巧讲解 (25分钟)

**贴片封装类型对比:**

```
ElectronBot涉及的封装类型:

1. LQFP-64 (STM32F405主控)
   ┌─────────────────────────────┐
   │  ┌───────────────────────┐  │
   │  │                       │  │
   │  │       芯片本体         │  │
   │  │       10×10mm         │  │
   │  │                       │  │
   │  └───────────────────────┘  │
   │  |||||||||||||||||||||||||   │  引脚间距: 0.5mm
   │  |||||||||||||||||||||||||   │  引脚数: 64 (每边16个)
   └─────────────────────────────┘  难度: 中等

2. QFN-32 (USB3300 PHY)
   ┌─────────────────────────┐
   │  ┌─┬─┬─┬─┬─┬─┬─┬─┐    │
   │  ├─┤           ├─┤    │  引脚在四周底部
   │  ├─┤           ├─┤    │  中间有裸露散热焊盘
   │  ├─┤  ┌─────┐  ├─┤    │  间距: 0.5mm
   │  ├─┤  │EPAD │  ├─┤    │  引脚数: 32 + 1(EPAD)
   │  ├─┤  └─────┘  ├─┤    │  难度: 高
   │  ├─┤           ├─┤    │
   │  └─┴─┴─┴─┴─┴─┴─┴─┘    │
   └─────────────────────────┘

3. TSSOP-20 (STM32F042舵机驱动)
   ┌───────────────────┐
   │ ┌───────────────┐ │
   │ │               │ │  引脚间距: 0.65mm
   │ │   芯片本体     │ │  引脚数: 20 (每边10个)
   │ │   6.5×4.4mm   │ │  难度: 容易
   │ └───────────────┘ │
   └───────────────────┘
```

**QFN焊接核心技巧:**

```
QFN焊接方法 -- 热风枪 + 锡浆法 (推荐):

步骤1: 准备焊盘
  ├── 用酒精清洁PCB焊盘
  ├── 在焊盘上均匀涂抹薄层锡浆
  │   （锡浆 = 锡粉 + 助焊剂, 呈灰色糊状）
  └── 锡浆厚度: 刚好覆盖焊盘即可, 不要太多!

步骤2: 放置芯片
  ├── 用镊子夹住芯片（注意方向！Pin 1圆点对齐PCB标记）
  ├── 轻轻放在焊盘上
  └── 在放大镜下微调位置，确保四周焊盘对齐

步骤3: 热风枪焊接
  ├── 热风枪设置: 280-300度C, 中低风速
  ├── 风嘴距离PCB约3-5cm
  ├── 均匀加热芯片区域（画圈移动，不要定住一点吹）
  ├── 约20-40秒后锡浆熔化
  ├── 芯片会"自动归位"（表面张力对齐）
  └── 再吹5-10秒确保焊料完全润湿

步骤4: 检查与补焊
  ├── 冷却1-2分钟
  ├── 用放大镜检查四周引脚
  ├── 如有连锡: 加助焊剂，用电烙铁拖焊清除
  ├── 如有虚焊: 在该引脚处加助焊剂，烙铁轻触焊盘边缘
  └── 如芯片歪了: 加热后用镊子轻推校正
```

**助焊膏使用注意事项:**
- 锡浆开封后要密封保存，干燥后会失效
- 涂抹用牙签或专用针管，不要用手直接接触
- 焊接后用酒精清洗残留助焊剂（助焊剂残留可能导致漏电）
- 锡浆的金属成分比例影响熔点（有铅63/37约183度C，无铅约217度C）

### 任务3.3: 舵机驱动板焊接 (50分钟)

**焊接顺序: 先小后大，先低后高**

Soldering order: small before large, low before tall.

**步骤:**

1. **认识舵机驱动板BOM（物料清单）**

   | 标号 | 元件 | 封装 | 数量 | 注意事项 |
   |------|------|------|------|----------|
   | U1 | STM32F042P6 | TSSOP-20 | 1 | 注意方向，Pin 1标记 |
   | Y1 | 8MHz晶振 | 3225 | 1 | 无极性 |
   | C1,C2 | 20pF电容 | 0402 | 2 | 晶振负载电容 |
   | C3 | 100nF电容 | 0402 | 4 | 去耦电容 |
   | C4 | 10uF电容 | 0603 | 1 | 电源滤波 |
   | R1 | 10kΩ电阻 | 0402 | 1 | BOOT0下拉 |
   | R2,R3 | 4.7kΩ电阻 | 0402 | 2 | I2C上拉 |
   | R4 | 电阻(地址配置) | 0402 | 1 | I2C地址设置 |
   | J1 | 4pin排针 | 2.54mm | 1 | I2C+电源连接 |
   | J2 | 3pin排针 | 2.54mm | 1 | 舵机信号连接 |

2. **焊接0402被动元件（电阻、电容）**
   ```
   0402元件焊接技巧 (尺寸仅1.0×0.5mm!):
   
   1. 在一个焊盘上加少量锡
   2. 用镊子夹住元件放到焊盘上
   3. 烙铁熔化预加锡，同时用镊子调整位置
   4. 撤烙铁，等2秒冷却
   5. 焊另一端
   6. 检查元件是否歪斜
   
   关键: 手要稳! 呼吸时暂停操作, 避免镊子抖动
   0402元件非常轻, 吹一口气都可能飞走
   ```

3. **焊接STM32F042P6 (TSSOP-20)**
   ```
   TSSOP-20焊接步骤:
   
   1. 在PCB上找到U1位置，确认Pin 1标记
   2. 在芯片的一个角焊盘上预加少量锡
   3. 用镊子放好芯片，Pin 1对齐PCB标记
   4. 烙铁熔化预加锡，固定芯片对角的引脚
   5. 放大镜下检查所有引脚是否对齐焊盘
   6. 如果没对齐，加热固定引脚的同时用镊子微调
   7. 对齐后逐个焊接其余引脚
   8. 如有连锡：加助焊剂，拖焊清除
   
   引脚间距0.65mm，比LQFP-64的0.5mm宽，
   相对容易焊接，适合作为第一个贴片IC练习
   ```

4. **焊接晶振和连接器**
   ```
   8MHz晶振(3225封装):
   - 无极性但接地焊盘要焊好（屏蔽效果）
   - 四个焊盘对齐即可
   
   排针连接器:
   - 先焊一个引脚固定
   - 检查排针是否垂直于PCB
   - 再焊其余引脚
   ```

5. **焊接质量检查**
   ```
   舵机驱动板检查清单:
   
   目视检查:
   [ ] STM32F042方向正确（Pin 1标记对齐）
   [ ] 所有0402元件无歪斜、无连锡
   [ ] 晶振焊接牢固
   [ ] 排针垂直于PCB
   [ ] 板面干净（无多余焊锡球）
   
   万用表检查（断电状态）:
   [ ] VDD(3.3V)到GND: 阻抗>1kΩ（不短路）
   [ ] I2C SDA到GND: 阻抗>4kΩ（上拉电阻在）
   [ ] I2C SCL到GND: 阻抗>4kΩ（上拉电阻在）
   [ ] 舵机PWM输出到GND: 不短路
   [ ] 电位器ADC输入到GND: 有阻值（电位器阻值）
   ```

**预期结果:**
- 完成至少1块舵机驱动板的全部焊接
- 万用表检查无短路
- 元件方向全部正确

**常见问题:**
- **0402元件太小看不清**: 使用放大镜或手机微距模式。操作时背部挺直，手臂支撑桌面
- **0402元件被烙铁吸走**: 烙铁头上沾了太多锡，产生的表面张力会把小元件吸起来。减少锡量
- **TSSOP连锡**: 加助焊剂后用干净烙铁头从中间向两边拖，连锡会自动分开
- **焊盘脱落**: 烙铁温度太高或加热时间太长（>5秒），焊盘铜箔脱离PCB基材。降低温度，缩短时间

## 下午: 主控板焊接 | Afternoon: Main Board Soldering

### 任务3.4: STM32F405主控焊接 (40分钟)

**这是今天最关键的焊接任务！LQFP-64有64个引脚，间距0.5mm，需要集中注意力。**

This is today's most critical soldering task! LQFP-64 has 64 pins at 0.5mm pitch -- full concentration required.

**步骤:**

1. **焊前准备（5分钟）**
   ```
   准备工作:
   1. 用酒精清洁主控板PCB焊盘
   2. 在放大镜下检查焊盘：无氧化、无损伤
   3. 在STM32F405的所有焊盘上薄涂一层助焊剂
   4. 检查芯片：确认型号为STM32F405RGT6
   5. 找到芯片的Pin 1标记（芯片上的小圆点或缺口）
   ```

2. **定位和固定（10分钟）**
   ```
   LQFP-64定位步骤:
   
   1. 在PCB上找到U1位置和Pin 1标记
   2. 在对角的两个焊盘上各加少量锡（如Pin 1和Pin 33）
   3. 用镊子夹住芯片两侧（不要碰引脚），放到焊盘上
   4. 仔细对齐:
      - Pin 1圆点对齐PCB丝印的Pin 1标记
      - 四边引脚均匀覆盖对应焊盘
      - 芯片居中于焊盘区域
   5. 先焊Pin 1引脚（固定一角）
   6. 放大镜下检查对齐
   7. 焊对角的Pin 33引脚（固定另一角）
   8. 再次检查对齐——还有机会调整!
   9. 如果歪了: 加热两个固定引脚，镊子轻推校正
   ```

3. **拖焊全部引脚（15分钟）**
   ```
   拖焊法焊接LQFP-64:
   
   1. 在所有引脚上涂助焊剂（覆盖整排引脚）
   2. 烙铁头带一小滴锡
   3. 烙铁头以45度角接触引脚根部
   4. 从引脚一端匀速拖向另一端（约2秒/排）
   5. 助焊剂会帮助锡均匀分布，连锡自动分离
   6. 每边引脚重复2-3次
   7. 四边共需约8-12次拖焊
   
   拖焊要点:
   - 速度要匀速，不要太慢（过热）也不要太快（锡不均匀）
   - 烙铁头保持干净，每次拖焊前擦拭
   - 如果某处锡太多，用吸锡带吸掉后重新拖焊
   - 不要用力压引脚（会弯曲甚至折断引脚）
   ```

4. **底部散热焊盘（如适用）**
   ```
   部分LQFP芯片有底部散热焊盘(Exposed Pad):
   - 在底部焊盘上涂锡浆
   - 用热风枪从背面加热
   - 或者在PCB背面散热过孔处用烙铁加热
   
   STM32F405RGT6 (LQFP-64) 通常没有底部焊盘
   但仍需确认具体封装版本
   ```

5. **焊接后检查（10分钟）**
   ```
   LQFP-64检查清单:
   
   放大镜检查（逐边）:
   [ ] 上边16个引脚: 无连锡、无虚焊
   [ ] 右边16个引脚: 无连锡、无虚焊
   [ ] 下边16个引脚: 无连锡、无虚焊
   [ ] 左边16个引脚: 无连锡、无虚焊
   
   万用表蜂鸣档检查:
   [ ] 3.3V(VDD)到GND(VSS): 不短路 (阻抗>100Ω)
   [ ] NRST到3.3V: 上拉电阻存在
   [ ] BOOT0到GND: 下拉电阻存在
   [ ] HSE晶振两个引脚: 不短路
   
   如发现连锡:
   1. 在连锡处涂助焊剂
   2. 用干净烙铁头（不带锡）从中间向外拖
   3. 如果拖不开，用吸锡带吸掉多余锡，重新拖焊
   ```

**预期结果:**
- STM32F405焊接到位，方向正确
- 所有64个引脚无连锡
- 电源到地无短路

**常见问题:**
- **引脚对不齐**: 先固定两个对角引脚，再微调。不要急，这一步决定了后续所有引脚的质量
- **大面积连锡**: 不要用更多锡去"冲开"——那只会让情况更糟。用吸锡带吸掉，加助焊剂重新拖焊
- **某几个引脚虚焊（焊锡没有润湿焊盘）**: 在该引脚处加一滴助焊剂，用烙铁头轻轻触碰引脚末端
- **芯片放反了**: 必须拆下来重新焊。用热风枪加热整颗芯片，等锡全部熔化后用镊子取下。千万不要硬撬

### 任务3.5: USB3300 QFN-32焊接 (35分钟)

**这是今天最难的任务！QFN-32封装的引脚藏在芯片底部，无法直接看到焊点。**

This is today's hardest task! QFN-32 has pins hidden under the chip body -- you cannot directly see the solder joints.

**步骤:**

1. **焊前准备**
   ```
   QFN焊接必须使用锡浆+热风枪法:
   
   1. 用酒精清洁USB3300焊盘区域
   2. 检查焊盘:
      - 四周32个信号焊盘
      - 中间1个大的裸露散热焊盘(EPAD)
   3. 准备锡浆: 用牙签挑少量锡浆
   ```

2. **涂锡浆**
   ```
   锡浆涂抹技巧:
   
   1. 用牙签在四周焊盘上涂薄层锡浆
      - 每个焊盘只需要极少量的锡浆
      - 锡浆应该刚好覆盖焊盘表面
   2. 在中间EPAD上也涂锡浆
      - 这是散热焊盘，连接GND
      - 锡浆量比周围焊盘稍多（需要导热）
   3. 不要涂太多！锡浆太多会导致芯片浮高或连锡
   ```

3. **放置芯片**
   ```
   QFN-32放置:
   
   1. 用镊子夹住芯片两侧
   2. 注意方向:
      - USB3300的Pin 1标记（小圆点）对齐PCB的Pin 1标记
      - 芯片文字方向与PCB丝印方向一致
   3. 轻轻放在焊盘上
   4. 放大镜下检查四周是否对齐
      - QFN的焊盘很短，对齐精度要求更高
   ```

4. **热风枪焊接**
   ```
   热风枪参数:
   - 温度: 280-300度C
   - 风速: 中低（2-3档，太大会吹跑芯片）
   - 风嘴: 小圆嘴
   
   操作:
   1. 热风枪距离PCB 3-5cm
   2. 以芯片为中心画圈均匀加热
   3. 约20-30秒后锡浆开始熔化
   4. 观察芯片: 如果芯片微微"浮动"然后"归位"
      说明锡已全部熔化，表面张力完成自动对齐
   5. 继续加热5秒确保充分润湿
   6. 移开热风枪，等待自然冷却（不要吹气加速！）
   
   冷却约2分钟后:
   7. 用镊子轻轻推芯片侧面
      - 如果纹丝不动 = 焊接良好
      - 如果能推动 = 可能虚焊，需要重新加热
   ```

5. **检查与补焊**
   ```
   QFN焊接检查:
   
   放大镜下检查四周:
   [ ] 每个引脚侧面能看到焊锡"脚印"（锡从芯片底部延伸到侧面）
   [ ] 无焊锡球（锡浆太多导致）
   [ ] 芯片贴合PCB，无明显翘起
   [ ] 芯片方向正确
   
   万用表检查:
   [ ] USB3300 VDD(3.3V)到GND: 不短路
   [ ] USB3300 VDDIO(2.5V)到GND: 不短路
   [ ] ULPI信号线到GND: 不短路
   [ ] USB DP到DM: 不短路
   
   常见补焊:
   - 四周引脚连锡: 加助焊剂，烙铁轻触连锡处
   - 底部虚焊: 只能重新热风加热
   - 锡球: 用镊子移除，加助焊剂清洗
   ```

**常见问题:**
- **芯片被热风吹跑了**: 风速太大。降低风速，也可以用耐高温胶带固定芯片对角
- **四周引脚全部连锡**: 锡浆涂太多了。用吸锡带+助焊剂清除，重新涂更薄的锡浆
- **芯片浮高（不贴合PCB）**: 中间EPAD锡浆太多。重新加热，用镊子轻轻下压
- **方向焊反了**: 只能热风枪加热取下，清理焊盘后重来

### 任务3.6: GC9A01屏幕焊接 (15分钟)

```
GC9A01屏幕焊接:

方式一: FPC连接器法（如果屏幕是FPC排线接口）
  1. 焊接PCB上的FPC连接器（0.5mm间距）
  2. 插入屏幕FPC排线
  3. 锁紧FPC座卡扣

方式二: 直接焊接法（如果屏幕引脚直接焊接）
  1. 屏幕排线很细，需要极其小心
  2. 先在PCB焊盘上预加锡
  3. 放置屏幕，对齐引脚
  4. 用烙铁逐个焊接（温度调低到280-300度C）
  5. 不要过度加热——屏幕的FPC基材容易受热变形

注意: 屏幕是易损件，操作时不要按压屏幕表面
```
