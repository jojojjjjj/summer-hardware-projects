# 最终展示要求 | Final Presentation Requirements

## 概述 | Overview

最终展示是项目的重要环节，学员将向老师和同学展示他们的"小智同学"智能音箱项目。

> The final presentation is an important part of the project, where students present their "Xiao Zhi" smart speaker project to teachers and classmates.

---

## 展示时间 | Presentation Time

**Day 12 下午 | Day 12 Afternoon**
- 每人 15-20 分钟
- 包含演示和问答

---

## 展示内容 | Presentation Content

### 1. 项目概述 (3分钟) | Project Overview (3 min)

**内容要求：**
- 项目名称和定位
- 核心功能介绍
- 技术栈概述
- 个人贡献说明

**PPT 页数：** 2-3 页

### 2. 功能演示 (8分钟) | Feature Demo (8 min)

**必演功能：**
- [ ] 语音对话（至少 3 轮）
- [ ] 唤醒词激活
- [ ] 插件功能（至少 2 个）

**选演功能：**
- 连续对话
- 特殊功能
- 错误处理

**演示技巧：**
- 提前测试所有功能
- 准备备用方案
- 控制演示节奏

### 3. 技术讲解 (5分钟) | Technical Explanation (5 min)

**讲解内容：**
- 系统架构图
- 核心技术点
- 遇到的挑战
- 解决方案

**PPT 页数：** 3-4 页

**重点讲解：**
1. **STT 模块**
   - Whisper 原理
   - 模型选择
   - 优化方法

2. **LLM 模块**
   - API 调用
   - 提示词设计
   - 对话管理

3. **TTS 模块**
   - Edge TTS 使用
   - 参数调整
   - 音质优化

### 4. 项目总结 (2分钟) | Project Summary (2 min)

**内容要求：**
- 完成情况
- 学习收获
- 遇到的问题
- 未来改进

**PPT 页数：** 1-2 页

### 5. 问答环节 (2分钟) | Q&A (2 min)

**准备回答的问题：**
1. 为什么选择这个技术栈？
2. 遇到的最大挑战是什么？
3. 如何优化响应时间？
4. 未来有什么改进计划？

---

## PPT 要求 | PPT Requirements

### 整体要求
- **总页数：** 10-15 页
- **风格：** 简洁、专业
- **语言：** 中英双语

### 必含页面
1. **封面页** (1页)
   - 项目名称
   - 学员姓名
   - 日期

2. **项目概述** (2-3页)
   - 项目简介
   - 功能列表
   - 技术栈

3. **系统架构** (1-2页)
   - 架构图
   - 模块说明
   - 数据流程

4. **核心功能** (3-4页)
   - STT 功能
   - LLM 功能
   - TTS 功能
   - 插件系统

5. **技术亮点** (2-3页)
   - 创新点
   - 难点攻克
   - 性能优化

6. **项目总结** (1-2页)
   - 完成情况
   - 学习收获
   - 未来展望

### PPT 模板

```markdown
# 封面页
## 小智同学智能音箱 | Xiao Zhi Smart Speaker
### 学员姓名 | Student Name
### 2026-05-06

---

# 项目概述 | Project Overview

## 项目简介 | Introduction
- 一个基于树莓派的智能语音助手
- A voice assistant based on Raspberry Pi

## 核心功能 | Core Features
1. 语音识别 (STT)
2. 智能对话 (LLM)
3. 语音合成 (TTS)
4. 唤醒词检测

## 技术栈 | Tech Stack
- 硬件：Raspberry Pi 4
- 语言：Python
- AI模型：Whisper, DeepSeek/OpenAI API
- TTS：Edge TTS
...
```

## 常见问题准备 | Common Questions

### 技术问题
1. **为什么选择 Whisper？**
   - 开源免费
   - 中文支持好
   - 准确率高

2. **为什么选择 Edge TTS？**
   - 免费使用
   - 音质自然
   - 中文支持好

3. **如何优化响应时间？**
   - 使用小模型
   - 异步处理
   - 缓存结果

### 项目问题
1. **最大的挑战是什么？**
   - 音频处理
   - API 调用
   - 系统集成

2. **学到了什么？**
   - Python 编程
   - AI 模型使用
   - 硬件组装
   - 项目管理
