## 常见问题 | Common Issues

### Q1: 对话响应太慢怎么办？

**A:** 优化建议：
1. 使用更小的 Whisper 模型
2. 减少 max_tokens
3. 使用更快的 API 提供商

### Q2: 唤醒词不工作怎么办？

**A:** 检查：
1. API Key 是否正确
2. 麦克风是否工作
3. 灵敏度是否合适

### Q3: 硬件连接问题？

**A:** 参考：
- `hardware/troubleshooting.md`
- 检查接线
- 测试音频设备
