## 组装流程 | Assembly Process

### 第 1 步：安装 MicroSD 卡

1. 找到树莓派背面的 MicroSD 卡槽
2. 将 MicroSD 卡金属面朝向树莓派
3. 轻轻推入直到卡住
4. 确认安装牢固

```
正确方向：
┌─────────────────┐
│  树莓派主板      │
│  ┌──────┐       │
│  │SD卡  │  ← 金属面朝上
│  └──────┘       │
└─────────────────┘
```

---

### 第 2 步：安装外壳（可选）

#### 选项 A：纸盒外壳

1. 找一个合适的硬纸盒
2. 在侧面开孔：
   - USB 接口孔（2-3 个）
   - 3.5mm 音频孔
   - USB-C 电源孔
   - 散热孔（多个小孔）

3. 将树莓派放入纸盒
4. 用胶带固定位置

#### 选项 B：购买外壳

1. 打开外壳包装
2. 按照说明书组装
3. 将树莓派安装到底座
4. 盖上上盖

#### 选项 C：3D 打印外壳

1. 下载 STL 文件
2. 3D 打印外壳
3. 组装外壳
4. 安装树莓派

---

### 第 3 步：连接音频设备

1. **连接麦克风：**
   - 将 USB 麦克风插入 USB0 接口
   - 确保连接牢固

2. **连接喇叭：**
   - 将 3.5mm 音频线插入 3.5mm 接口
   - 连接到喇叭
   - 确保连接牢固

```
连接示意图：
     [USB 麦克风]
          │
          ↓
    ┌────────┐
    │USB0   │
    │USB1   │
    │  ...  │
    └────────┘
         │
         ↓
    ┌────────┐
    │树莓派  │
    │        │
    │3.5mm  │←──[喇叭]
    └────────┘
```

---

### 第 4 步：首次启动配置

1. **连接显示器：**
   - 使用 HDMI 线连接显示器
   - 连接到 HDMI0 接口

2. **连接键盘鼠标：**
   - USB 键盘插入 USB 接口
   - USB 鼠标插入 USB 接口

3. **连接电源：**
   - 最后连接 USB-C 电源
   - 等待启动

4. **首次配置：**
   ```bash
   # 更新系统
   sudo apt update && sudo apt upgrade -y

   # 配置音频
   sudo raspi-config
   # System Options → Audio → Force 3.5mm jack

   # 启用 SSH（可选）
   # Interface Options → SSH → Enable
   ```

---

### 第 5 步：验证硬件

1. **测试麦克风：**
   ```bash
   # 录制 5 秒音频
   arecord -d 5 -f cd test.wav

   # 播放录音
   aplay test.wav

   # 如果听到自己的声音，麦克风工作正常
   ```

2. **测试喇叭：**
   ```bash
   # 播放测试音
   speaker-test -t wav -c 2

   # 如果听到测试音，喇叭工作正常
   ```

3. **测试音频路由：**
   ```bash
   # 检查默认设备
   aplay -l
   arecord -l

   # 应该能看到音频设备
   ```

---

### 第 6 步：最终组装

1. **整理线缆：**
   - 使用扎带整理线缆
   - 避免线缆缠绕

2. **固定位置：**
   - 将组件固定在外壳内
   - 确保所有接口可访问

3. **散热考虑：**
   - 不要堵塞散热孔
   - 如有需要，安装散热片

4. **最终测试：**
   - 连接电源
   - 确认所有功能正常

## 常见问题 | Common Issues

### Q1: 树莓派无法启动

**可能原因：**
- SD 卡未正确安装
- SD 卡损坏
- 电源不足

**解决方法：**
1. 重新安装 SD 卡
2. 尝试另一个 SD 卡
3. 检查电源是否为 5V 3A

### Q2: 麦克风无法工作

**可能原因：**
- USB 接口问题
- 驱动问题

**解决方法：**
1. 尝试其他 USB 接口
2. 检查 `lsusb` 是否识别
3. 安装音频驱动

### Q3: 喇叭没有声音

**可能原因：**
- 音频输出设置错误
- 音量太低
- 喇叭损坏

**解决方法：**
1. 使用 `raspi-config` 设置音频输出
2. 使用 `amixer` 调整音量
3. 测试喇叭是否损坏
