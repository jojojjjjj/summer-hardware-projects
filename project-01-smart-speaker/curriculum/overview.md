# 课程总纲 | Course Overview

## 课程信息 | Course Information

| 项目 | 内容 |
|------|------|
| **课程名称** | 智能音箱 "小智同学" |
| **课程周期** | 12 天（每天 6-8 小时） |
| **适合年级** | 高中二年级+ |
| **先修要求** | 基础 Python 知识 |
| **教学方式** | 项目驱动学习 |

## 12天课程安排 | 12-Day Schedule

### 第1周：基础准备 | Week 1: Foundation

| 日期 | 主题 | 核心内容 |
|------|------|----------|
| **Day 1** | 项目介绍 + 环境搭建 | 树莓派系统安装、Python环境配置、项目结构理解 |
| **Day 2** | Python 温故知新 | Python基础语法回顾、pip包管理、虚拟环境 |
| **Day 3** | 音频基础 | 音频采样原理、录音与播放、PyAudio使用 |
| **Day 4** | 语音转文字（ASR） | Whisper模型原理、安装与使用、语音识别实践 |
| **Day 5** | 文字转语音（TTS） | TTS原理、Edge TTS使用、语音合成实践 |
| **Day 6** | LLM API 调用 | API调用基础、DeepSeek/OpenAI API、对话逻辑 |

### 第2周：核心功能实现 | Week 2: Core Features

| 日期 | 主题 | 核心内容 |
|------|------|----------|
| **Day 7** | 语音对话 MVP | STT→LLM→TTS完整串联、基础对话循环实现 |
| **Day 8** | 优化与调试 | 延迟优化、错误处理、日志系统 |
| **Day 9** | 唤醒词检测 | 唤醒词原理、pvporcupine使用、实时监听 |
| **Day 10** | 硬件组装 | 外设连接、外壳制作、系统集成测试 |
| **Day 11** | 功能扩展 | 天气查询、音乐播放、自定义指令 |
| **Day 12** | 项目优化 + 演示准备 | 代码优化、演示准备、最终展示 |

## 学习路径 | Learning Path

```
Day 1-3: 基础准备
    └── 环境搭建 + Python复习 + 音频基础

Day 4-6: 模块学习
    ├── ASR (Whisper)  → 让机器"听"
    ├── TTS (Edge TTS) → 让机器"说"
    └── LLM API        → 让机器"思考"

Day 7-9: 系统集成
    └── 完整对话循环 + 唤醒词功能

Day 10-12: 完善优化
    └── 硬件组装 + 功能扩展 + 演示准备
```

## 每日课程结构 | Daily Class Structure

每个每日课程文件（day-01.md 到 day-12.md）都包含：

1. **学习目标** (Learning Objectives) - 中英双语
2. **前置准备检查** (Prerequisites Check)
3. **为什么学这个** (Why Learn This) - 真实应用解释
4. **具体任务** (Tasks) - 2-3个实际操作任务
5. **今日作业** (Today's Assignment)
6. **明日预告** (Preview of Tomorrow)
7. **参考资源** (References)

## 评估节点 | Assessment Milestones

| 节点 | 时间 | 内容 |
|------|------|------|
| **Week 1 Check-in** | Day 6 结束 | 环境搭建完成、三个模块独立运行 |
| **Week 2 Check-in** | Day 9 结束 | MVP完成、唤醒词集成 |
| **Final Presentation** | Day 12 | 完整演示 + PPT展示 |

## 关键里程碑 | Key Milestones

- **Day 3:** 能够录制和播放音频
- **Day 6:** 能独立使用ASR、TTS、LLM三个模块
- **Day 7:** 完成第一次语音对话循环
- **Day 9:** 唤醒词功能工作
- **Day 12:** 完整可演示的智能音箱

## 预期成果 | Expected Outcomes

完成课程后，学员将能够：
1. 理解语音助手的工作原理
2. 使用开源API和模型构建智能应用
3. 进行基础硬件组装和接线
4. 编写模块化的Python代码
5. 进行技术演示和文档编写

After completing the course, students will be able to:
1. Understand how voice assistants work
2. Build intelligent applications using open-source APIs and models
3. Perform basic hardware assembly and wiring
4. Write modular Python code
5. Conduct technical presentations and documentation
