# Arduino 库依赖 | Arduino Library Dependencies

## 需要安装的库 | Required Libraries

以下库可以通过 Arduino IDE 库管理器 (Ctrl+Shift+I) 安装：

| 库名 | 安装方式 | 用途 |
|------|---------|------|
| U8g2 | 库管理器搜索 "U8g2" (by olikraus) | OLED 显示驱动 |
| ArduinoJson | 库管理器搜索 "ArduinoJson" (by Benoit Blanchon) | JSON 数据解析 |

以下库需要手动安装（GitHub 下载）：

| 库名 | 下载地址 | 用途 |
|------|---------|------|
| wifi_link_tool | https://github.com/bilibilifmk/wifi_link_tool | WiFi 配网工具 |
| Time | 已包含在 software/src/libraries/Time/ | 时间管理 |

## 手动安装步骤

### wifi_link_tool
1. 访问 https://github.com/bilibilifmk/wifi_link_tool
2. 点击 Code → Download ZIP
3. Arduino IDE → 项目 → 加载库 → 添加 .ZIP 库
4. 选择下载的 ZIP 文件

### Time 库
1. 本项目的 `software/src/libraries/Time/` 文件夹
2. 复制到 Arduino 的 libraries 目录：
   - Windows: `Documents\Arduino\libraries\Time\`
   - macOS: `~/Documents/Arduino/libraries/Time/`

## ESP8266 开发板支持

在 Arduino IDE 中：
1. 文件 → 首选项 → 附加开发板管理器网址
2. 添加：`https://arduino.esp8266.com/stable/package_esp8266com_index.json`
3. 工具 → 开发板 → 开发板管理器
4. 搜索 "ESP8266" → 安装 "esp8266 by ESP8266 Community"
